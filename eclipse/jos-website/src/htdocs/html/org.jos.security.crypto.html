Home of the JOS cryptographic security provider.

Description

A free cryptography security provider, compatible with Sun's Java Cryptography Extensions (JCE).

The library contains implementations of: SHA-1, MD5, IDEA, CAST5, DES, 2DES, 3DES, RC2, RC4, RSA, Diffie-Hellman, ElGamal, DSS
