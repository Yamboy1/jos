JExplorer 1.1Quick view (Feb 2001)

  • Renames folders directly in the folders tree.
  • Refreshes the folders tree when the file list changes its content
  • Represents ZIP and JAR files as folders.
  • Context menu in the file list.
  • Support for external viewers (as plugins)
  • HTML and Text pluggin
  • Context menu in the tree
  • Detailed view
  • External viewers can read zip files
  • Image pluggin
  • Apply filters to the file list (.jar or J.)
  • Revision of the interface QuickView
  • Vertical and horizontal viewers
  • Film view
  • Configuration file: filetype.properties
  • File Properties window
JExplorer 1.0 (0.7.294) Quick view (Aug 2000)
  • Brows your PC file system.
  • Copy files and/or folders.
  • Delete files and/or folders.
  • Rename files or folders.
  • Big icon view or brief view.
  • Internationalization support.
  • Gnome Look (toolbar and icons).

For more information see JExplorer site
