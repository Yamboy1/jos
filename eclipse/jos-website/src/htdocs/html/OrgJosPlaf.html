Project goals

  • Design and implement a new pluggable look-and-feel to accompany JOS.
    • Allow themes for colors, behavior, sound
    • Incorporate the best and most intuitive features of other UIs
    • Provide mini-applications for L&F/theme configuration
  • Create emulation L&F's of popular UIs

Project Tasks

  • Create UI delegates (the V&C of MVC)
  • Design themes
    • Develop standard theme interface(s)
    • Develop individual themes
    • Develop theme configuration apps
  • Create UI resources: icons, sounds, etc.
  • Investigate legal issues for emulation of other UIs
