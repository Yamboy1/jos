Sun Microsystems Laboratories' Note on Distribued Computing
http://www.sunlabs.com/technical-reports/1994/abstract-29.html
Jim Waldo, Geoff Wyant, Ann Wollrath, and Sam Kendall discuss Sun's position on distribued computing. This paper led to many design decisions for RMI.

Open Distributed Systems
http://www.cs.ucl.ac.uk/staff/jon/ods/ods.html
A complete online book by Jon Crowcroft. An introduction to open distributed systems with an emphasis on the standards procedure. - IanDavis

Parallel and Distributed Computing, an Overview of Current Trends
http://www.dcss.mcmaster.ca/~gilbert/outline.html
by Dave Gilbert - An interesting essay on some of the concepts and obstacles to parallel and distributed computing development - IanDavis

The Berkeley NOW Project
http://now.cs.berkeley.edu/
A project to utilise networks of workstations to create distributed supercomputers. Good bibliography. - Ian Davis

Legion
http://www.cs.virginia.edu/~legion/
Project to build services to enable the creation of a world-wide virtual computer. - Ian Davis

WebOS
http://now.cs.berkeley.edu/WebOS/index.html
A project to create a global operating system based around the world wide web. Features include a Web File System that uses the URL namespace, and secure remote execution.
