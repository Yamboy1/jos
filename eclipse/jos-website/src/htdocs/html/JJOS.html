Introduction

JJOS refers to both the micro-kernel/JVM combination we (JohnMorrison and ToddMiller) call the native-code runtime, and specifically to the (micro-)kernel when used in reference with decaf.

23 August 2001

Since February 1999, a lot of effort has gone into making a binary edition of JJOS and keeping at least one version running. The binary edition is available on the DownloadJOS page. Versions 1f, 1g, 1h, 1i and 2a of JJOS have been constructed. All source code for this project has been moved to JJOSCvsOnSourceForge. -- GilbertHerschberger

20 February 1999

JJOS has been on hold awaiting "gen two" of decaf, but it should be stable.
