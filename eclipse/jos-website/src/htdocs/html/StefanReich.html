Hi everybody,

I quit the JOS project more than a year ago. I had started to develop a FileSystem architecture. The code (in its unfinished state, as it was back then) is available at http://www.drjava.de/jos/jos-filesystem-4-preview.jar . It contains a working pure java FAT driver! Who knows, maybe there will be some use for it somewhere on this huge planet. Some day.

Some concluding remarks... I would love to see the JOS project become a success. But I don't want to be part of it anymore. I'm frustrated with the lack of real progress. The point where an open-source OS takes off is when it is bootable. Then it will attract people from all over the world who are eager to port their favorite application to the new platform.

Bring JOS to that point, if you can!

Best wishes to all of you, it was much, much more than a nice time to work on this thing with you.

Stefan
