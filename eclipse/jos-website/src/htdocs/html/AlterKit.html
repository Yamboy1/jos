Application: AlterKit

Type: Utility

Package: org.jos.io.alterKit

Objective: Used to convert objects thru AlterMethods into byte arrays for streaming purposes (see AlterKitInfo ).

This utility contains the following features:

  • AlterKit
    • converts objects thru AlterMethods into byte arrays
    • Allows AlterMethod to be changed
  • AlterMethod
    • Allows cascading AlterMethods
  • DefaultAlterMethod
    • AlterMethod used for straight conversion of objects to byte arrays
  • FlaterAlterMethod
    • Uses Inflater/Deflater compression
    • Setting for compression level and strategy
  • GZIPAlterMethod
    • Uses GZIP compression
    • Setting for compression level and strategy
