Hi folks,

I feel strongly that the world needs as many high-quality open-source computer resources as it can get. The JOS project seems like exactly the right idea at exactly the right time. I hope it comes to fruition and I'd like to do what I can to help it do so.

I have fair few years experience in academia and the software industry using Cobol, Fortran, C/C++ etc. on PCs, minis and mainframes. I'm currently getting up to speed on Java and design patterns. Hopefully I can contribute to something to the JOS project as my understanding of those subjects improves.

I'm particularly interested in the Object-based Filesystem and GUI projects but I'd be willing to work on whatever seems usefull to the project as a whole. (Cleanroom issues might preclude my working on JOS ORB stuff.)

I can be reached at lrjos@ibm.net
