The general idea behind this project is to create a platform-independent, project-independent revision control system. The entire system is accessed through online web pages that are served up by Java servlets. The system allows for any kinds of projects to be maintained, such as source code for software, documentation, or anything else that can be handled by a file-based revision control system. Take a look at the specs and let me know if you have any ideas.

-- RyanSullivan, 9.2.98
