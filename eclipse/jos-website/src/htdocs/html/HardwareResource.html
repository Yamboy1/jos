A hardware resource is a part of the physical hardware.

Examples are:

  • DMA channels
  • Interrupts (including IRQ lines)
  • I/O Address Ports
  • Memory Blocks
The hardware resources are thought of as limited-access objects, so that only a handful of objects with permissions may gain access and utilize them.

The HardwareResourceManager maintains a list of allocated resources, and divies out the resources upon a valid request.
