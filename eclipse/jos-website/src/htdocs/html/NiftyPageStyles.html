What is this?

These are HTML templates that JOS members can use to quickly create project pages. For example, if I want to create a personal home page on the Wiki server dedicated to a JOS project that I'm working on, I would select the PersonalHomePage layout below, copy it to my computer's clipboard, and then paste it into the 'Edit Page' box for the page I wish to create.

If you are familiar with the concept of patterns, then these are WikiWebPagePatterns.

Which templates are available?

Here are some currently available templates. Feel free to add your own.

PersonalHomePage - Use this template to create personal home pages to track projects. [Courtesy of TimTaylor]

ProjectOne - This template is a straightforward description of a project's goals, achievements, & contact information. [Courtesy of JeroenVanGelderen]

NoticeBoard - This is a notice-board table that can be added to a project's homepage. [Courtesy of JeroenVanGelderen]

ProjectPackages - This describes all of the classes & smaller packages within a larger project's package. The example given is called PersonalWeb, which is not a real working package.

PatternTemplate - This template is for defying a pattern [via DigiGod]
