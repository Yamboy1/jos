MaintenancePages

Article contributed by GilbertHerschberger (12 February 2002).

Introduction

Due to circumstances beyond our control, our well-known Internet sites are missing. Countless links to these sites are now broken and need to be reworked.

JOS Project homepage

For now, the following link will be our substitute for jos.org, www.jos.org and wiki.jos.org.
  • http://jos.sourceforge.net/
While it is actually the homepage for JOS Wiki, it will have to serve double duty until we get our DNS entries straightened out.

Accessing JOS Project website

You are no longer required to modify your hosts file to see the older JOS Project website. This archive provides a snapshot of the website previously found on the jos.org server.
  • http://cjos.sourceforge.net/archive/
Only the most critical links to jos.org should be updated. When you click the Home link on a JOS Wiki page, for example, you should see this archive.

Info

An archive of the Info section of our website is found here:
  • http://cjos.sourceforge.net/archive/info/

DevZone

An archive of the DevZone section of our website is found here:
  • http://cjos.sourceforge.net/archive/dev/

Logos

We have maintained a collection of graphic images on the jos.org server. These images are included in the archive of the JOS Project website. Meanwhile, pages on JOS Wiki and elsewhere should use the following location to include our images.
  • http://cjos.sourceforge.net/archive/media/

On-Line Version of JOS Technical Edition

The JOS Technical Edition has a permanent new home on the web. It is only available here. It is no longer available in the redist directory on the jos.org server--even if you modify your hosts file.
  • http://cjos.sourceforge.net/redist/

Mirror of CjOS Project website

Here is a new feature. If you would like to download the on-line version of the JOS Technical Edition and add it to your own web server, this directory contains all of the necessary archives. Each piece of the product must be downloaded separately.
  • http://cjos.sourceforge.net/mirror/

Mailing list archives from jos.org

Since the JOS Technical Edition contains a snapshot of it, the mailing list archives do not need to converted. No version of mailman needs to be installed.
  • Starting at the homepage of the JOS Technical Edition, follow the Mailing List Archives link, or
  • http://cjos.sourceforge.net/redist/mail/

Source Tree

Since the JOS Technical Edition contains a snapshot of it, the Source Tree does not need to be converted.
  • Starting at the homepage of the JOS Technical Edition, follow the Source Tree link, or
  • http://cjos.sourceforge.net/redist/srcserv/

New JOS Project website

Once the archive has been reworked so that its links point to the right locations, a new JOS Project website will be started. The location of this website has not yet been determined. Our new homepage will look very much like the older one. Additional news items will be added. Some blurb explaining our recent DNS problems will be written. Likely, a link to this article will be provided if someone wants more information.
  • Our constitution is now available at http://cjos.sourceforge.net/archive/constitution.html
  • And our policy book is now available at http://cjos.sourceforge.net/archive/policybook.html
-- GilbertHerschberger (15 February 2002)
