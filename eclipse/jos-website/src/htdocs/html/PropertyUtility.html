Application: PropertyUtility

Type: Utility

Package: org.jos.util

Source Code: http://www.jos.org/source/org/jos/util/PropertyUtility.java.html

Objective: Used to save and retrieve customised property settings, usually per user.

This utility contains the following features:

  • Loads and save properties from file
  • Saves and retrieves items from properties file
