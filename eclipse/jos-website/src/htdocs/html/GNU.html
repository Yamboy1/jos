GNU is Not Unix

  • http://www.gnu.org/
  • http://www.gnu.org/software/java/java-software.html
  • http://www.fsf.org/
The GNU project, started by the FSF intended and still intends to create a completely free UNIX incarnation (The GNU system), based on the GNU Hurd Kernel. A variant of the GNU system, basing on the Linux Kernel is already in wide use, commonly referred to as Linux while others prefer the name GNU/Linux.
