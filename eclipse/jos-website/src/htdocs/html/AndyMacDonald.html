I little about me, hmmm...

I've spent the last seven years building distributed VR systems for one group or another, some for research orgs, some commercial. When I'm not getting paid to build these systems, I do it on my own, which is what I'm up to now. It was my quest to find a free, open, Java-friendly OS to build my latest project on that led me to JOS.

I've really fallen in love with the Java language over the last couple of years, and a number of pet projects that I've struggled to do in C, C++ or Lisp seem to be much easier with it. Built in multi-threading and synchronization, as well as RMI, seem to make it a natural candidate for most of the work I want to do.

I'm not sure exactly how much time I'll be able to devote to development, maybe just a few days a month, but wherever my skills can be used, I'll gladly employ them. Skills include:

  • 3D graphics
  • peripheral interfacing, both serial- and bus-based
  • user interface (3D and 2D)
  • real-time systems (I wrote a real-time executive a while ago, and I'd like to do more of that)
  • game development (who do you think pays for distributed VR?)
I think the JOS effort is fantastic. I'll keep checking in and pick up projects as I have the time.

--AndyMacDonald 05-MAR-1998
