The JOS Project 1998 First Quarter Status Report

1. OVERVIEW

What is JOS?
A Free Java Based Operating System ( FreeOS )
2. MAJOR ANNOUNCEMENTS
Wiki Web Enhancements
We have added revision control and support for other groups to host their Wiki on our site to collaborate with us.

...

Chat Client
Spin has donated a chat server at www.jos.org/chat

RTEMS evaluation and pending agreement.
Almost there...

JAS working relationship.

3. PROJECT STATUS
FileSystem
A working Pure Java File System has been developed:
  • Pluggable File System architecture (VFAT is here, ext2fs in preparation)
  • GUI test environment (JosExplorer)
  • Native disk drivers are in preparation
DriverInterface
We have started to develope a driver interface to allow for java base drivers.
Kernel
We hoped to have a prototype ready within a few day "Welcome to JOS"
APIStructure
