UserInterfaceDefinitions:

This page was created to aid in communication between the members of the user interface group and its mailing list by providing the definitive definition of words that have caused or may cause confusion. These are not the only meanings of some of these words, but unless otherwise noted, what they refer to in the UserInterfaceGroup or its mailing list.

UI - User Interface
The software and medium in which the user commands the system

PUI - Pluggable User Interface
A UI that "plugs" into the socket of JADE

GUI - Graphical User Interface
Any UI that uses graphics especially a TGUI

TUI - Textual User Interface
A windowing system made out of text charecters rather then graphics

TGUI - Traditional Graphical User Interface
Any traditrional windowing system such as Windows98 or MacOS

XUI - eXperimental User Interface
A UI based on new ideas such as voice recognition or 3D

CLI - Command Line Interface
Any UI based upon the input of text to a command processer

Terminal
A graphical emulation of a console, a CLI

Console
A text CLI

Shell
A framework or library consisting of the basic components required for all UI's, not to be confused with a CLI
