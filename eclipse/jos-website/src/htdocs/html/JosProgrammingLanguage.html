One of the issues unresolved in the JOS project is what programming language to use for our source code. Is the Java programming language possible? Some have strongly suggested C or C++. To share a codebase, we should all be agreed on its programming language.

Just like JosLicense, the following table helps you find out what others might prefer. Add yourself and, if possible, a reason why.

Name Preferred Language Short reason
GilbertHerschberger C/C++ A real system programming language should be used to build an operating system.
