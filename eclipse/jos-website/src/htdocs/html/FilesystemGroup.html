The FilesystemGroup (part of PersistenceGroup) aims to

  • design a driver spec to include drivers for hard disks, floppy drives etc. (MediumInterface)
  • design a driver spec to access file systems (ext2fs, FAT, ...) on the devices named above
  • design an API to let applications access extended file system functions (grant file permissions etc.) not covered by java.io.* classes
  • implement the drivers
Where to find...
  • working source code: FilesystemCode
  • a design proposal: FilesystemArchitecture
  • the discussion: FilesystemDiscussion

  • NicolaiGuba (ext2fs implementation)
