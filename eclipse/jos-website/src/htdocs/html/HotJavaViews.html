HotJavaViews is JavaSoft's first user environment that runs on top of JavaOS(tm). Taking advantage of the platform independence provided by Java, the upcoming release of HotJava Views 2.0 will be fully platform-independent, and will run on servers that contain the Java Virtual Machine.

The preceeding text was copied from http://www.javasoft.com/products/hotjavaviews/index.html -- BillRehm

2.0 is shipping, according to the page, to NC OEMS. More interesting is a demo applet (slightly buggy) and a whitepaper on the interface design, with appropriate illustrations. This is some beautiful stuff. They've eliminated the desktop metaphor almost completely, if not entirely. All menus are replaced with toolbars and drawers of buttons. IT looks really good.
