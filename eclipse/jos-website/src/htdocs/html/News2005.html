4 February 2005

About CjOS

Release 24 of CjOS 1.0 is available for immediate download. For more information, see also http://sourceforge.net/projects/cjos/ .

This release includes the Smart Cryptography plug-in for Smart API. Using a uniform resource identifier, your Java program can now

  • Decrypt while reading from a (password-protected) resource,
  • Encrypt while reading from a (plain-text) resource, and
  • Decrypt or encrypt while writing to a resource.
Protect resources like these with a secret password:
  • a Java resource (stored in a jar)
  • a Java properties file
  • an object stream
  • a data registry-compatible resource
  • an XML document
This release includes the Smart SSL plug-in. Using a uniform resource identifier, your Java program can now
  • Create a two-way SSL connection to an HTTP service,
  • Treat a Provider as a resource, and
  • Treat a keystore as a resource.
This release includes the Smart Security plug-in. Using a uniform resource identifier, your Java program can now
  • Treat a PKCS#8 private key as a resource.
This release includes the Private Key program. This program enables you to easily import a PKCS#8 private key into a JKS-compatible keystore.
