Jan, 1998 - The root package hierarchy seems mostly frozen now and first source code is released.

Nov 9, 1997 - The whole website moved to WikiClone, 263 topics, 208KB.

Oct 25, 1997 - WikiClone collaboration system online.

Oct 17, 1997 - Work begins on JOS Requirements document: JosRequirements

Oct 16, 1997 - jos.org created as home of the JOS Project.

Oct 14, 1997 - JOS consented to as name for the OS.

Oct 12, 1997 - java-os-project mailing list created. (Thank you MarkusPeter!)

Oct 3, 1997 - StefanBorg posts to JavaLobby JavaOS Forum asking if any of the JavaLobby members are interested in creating a free Java(tm) based OS.
