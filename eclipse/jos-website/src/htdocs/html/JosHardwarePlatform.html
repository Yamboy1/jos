One of the issues unresolved in the JOS Project is what hardware platform to use first. Is it possible to use Java-based hardware? Some have strongly suggested the ever-popular IBM PC compatible with an Intel 80386 or 80486 microprocessor. To run the same code, we must have a common hardware platform.

Just like JosLicense and JosProgrammingLanguage, the following table should help you contact others who want to work on the same hardware you do. Remember to add yourself here.

Person Preferred hardware Short reason
GilbertHerschberger IBM PC compatible + Intel 80486; other platforms should follow. Proven technology, upward compatibility, popularity, accessibility and cost of entry. Pick up a used '486 for $25. Use Linux g++ and Borland C++ 5.0.
