A replacement for WikiClone written in Java as a servlet.

This currently cannot replace WikiClone because we cannot run servlets at this time, but as soon as we can run servlets, this should serve as a replacement.

Note that there is still a need for someone to take over maintenance of WikiClone.

Requirements for this should be placed in WikiServletRequirements.

Please contact AveryRegier if you have interest in helping with this project.
