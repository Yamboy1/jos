Objective:

A global OS address book containing business names, business contacts and personal contacts

Should contain the following features:

  • Each contact will have the following
    • A name - be it company name, or personal name
    • An address
    • Multiple phone numbers (incl voice, fax and data distinction)
    • Multiple email addresses
Future goals:
  • Facilities to dial, fax or email any contact.
  • Full communications log of all incoming and outgoing connections.
  • A reference field for future use :)
Notes:
  • The framework for this project is pretty well already coded and its open for suggestions. The models are finished but I've had no such luck with the GUI ie I need help!!
  • I would like to implement an InfoBus rather than the JavaBeans Property listeners.
  • This project is a module for a much larger business application which has been in the making for 3 years. Problem is - its all in my head!!
--StefanBorg
