Application: PlainTextEditor

Type: Bean/Application

Package: org.jos.app.editor

Source Code: http://www.jos.org/source/org/jos/app/editor/PlainText.java.html

Objective: To produce simple plug-in bean text editor, specifically for unformatted text.

This bean contains the following features inherently:

  • Multiple Undo/Redo (pending)
  • Basic editing - cut, copy, paste and selectAll
  • Drag and Drop (pending)
  • Optional Line-wrapping, Automatic Word/Paragraph selection
  • Shortcut keys, Drop down menus, Toolbars
  • Pop-up menus (pending)
  • Selectable Font, size, etc and Tab width
  • Selectable text and background colours (pending)

This bean contains the following features optionally in the form of plug-ins:

  • File open, save, auto-save, close, exit (pending)
  • Insert File and Save As(pending)
  • Printing, Print Preview and Page Setup(pending)
  • Find and Replace(pending)
  • Case conversion [UPPERCASE, lowercase, TitleCase, Sentance case](pending)
  • Insert Date(pending)
  • Spelling/Thesaurus(pending)
  • Insert Address from AddressBook(pending)
