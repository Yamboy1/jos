Introduction

DaemonRunner is a central place to load daemons/services. This will let the admin only have to look in one place to configure the daemons. Instead of searching through startup scripts. :)

Much like the NT Service Manager. This will give the user the ability to quickly add a daemon. Configure it and disable/enable a daemon.

I have initial support for username/password so later when we figure out how we are going to do permissions, we can have daemons start as a specified ID.

The actual DaemonRunner works and I plan on beginning the GUI client to configure it immediately.

Comments/Thoughts welcome.
