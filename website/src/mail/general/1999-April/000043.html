I know we've been discussing using BeanShell many times before, but I felt like
argumenting for it now.
I peeped Pat's site (http://www.ooi.com/beanshell/) today, and dl:ed the latest
version (0.96 beta). It's so great. I can't understand why it's still pretty
unknown. Maybe it isn't. =P

It's GPL'ed now.
You can call any java class from inside of it, you can run it in interpreter
mode, console mode and server mode. The server mode is waay cool. You just
specify a port, and it launches a tiny httpd. When you connect to it with a
browser, it starts an applet and Whoop! You've got a bsh console.

Check it out.
