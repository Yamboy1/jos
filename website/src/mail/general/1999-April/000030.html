Some people have been seeing errors while checking out files under
jos/library/org/jos/server. This directory should not have existed and I
have now removed it.

You should now be able to checkout or update. If you still get errors,
try the following:

1. Remove your jos/library/org/jos/server directory.
2. Run checkout again (or cvs update -d if that's what you were trying
to do).

If that still doesn't work, try this:

1. Move your current source tree to a temporary directory.
2. Do a fresh checkout of the module.
3. Copy across any files you were editing from your temporary directory
to your new directory.

If you're still having problems, just send me an email.
