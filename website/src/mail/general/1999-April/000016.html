JOSers:

There is a new editorial in the Bite Size Java column on O'Reilly's site
that laments about the lack of a good Java Operating System.  We are
mentioned as heading in the right direction but not far enough along and too
slow developing.  He makes some good points about the memory savings and
advantages that can be gained from a Java OS with an integrated kernel/jvm.
(See http://jos.org/wiki/view/main/WhyMultiProcessJVM for my take on that.)

It is at: http://java.oreilly.com/news/newos_0499.html

Take a look, it is a good read.  Then write back here and let us all know
what YOU think!
