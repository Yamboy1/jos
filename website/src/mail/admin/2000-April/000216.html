How many boxes do we need?

We need one for "www.jos.org", which we already have thanks to Helmut.

We need one for "wiki.jos.org", which we already have thanks to
metamech.com. The Wiki and HTTP services should never run on the same
machine. A Wiki snapshot should be stored redundently on an HTTP service --
just in case the Wiki service goes down.

We need one for "ftp.jos.org". The FTP and CVS services should never run on
the same machine. A CVS snapshot should be stored redundently on an FTP
service -- just in case the CVS service goes down.

We need one for "cvs.jos.org". We almost have one thanks to SourceForge.
Both CVS and CVSWeb must either (1) run on one machine, or (2) share a file
server. I made a snapshot of JOS CVS on April 20, 2000. This snapshot does
not contain a revision history, only the latest revision.
