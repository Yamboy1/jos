Mailing lists are the primary means of communicating on critical issues.
Look at us now. We're using the administration mailing list to decide on a
very critical issue.

The mailman program does not have to run on the same machine with HTTP,
FTP, Wiki, CVS and CVSWeb, does it? Mailman might run on a separate
machine, called "mailman.jos.org". The machine's physical location is
irrelevant. The machine's IP address is irrelevant. Who owns the machine is
irrelevant.

And still, the "gateway" server must point to THE mailman application. As
an open source project, we cannot afford to cut our lines of communication.
Our mailing list should be important enough to put it on a separate machine.
