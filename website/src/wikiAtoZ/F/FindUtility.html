AboutSmartAPIUtilities

Article contributed by GilbertHerschberger (3 January 2001).

Introduction

The Find utility is part of Smart API Utilities. It searches for files by name.

Location

Enter the following location to start the Find utility. It works in the location field of both the UniversalBrowser and Program Browser.

run:program:org.jos.gchii.find.Find

Wildcard characters

The asterisk (*) and question mark (?) are wild card characters. While the asterisk matches any number of characters, the question mark matches exactly one.
