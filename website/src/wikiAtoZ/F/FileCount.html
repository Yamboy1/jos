AboutSmartAPIUtilities

Article contributed by GilbertHerschberger (10 October 2000).

Introduction

This utility is part of the Smart API Utilities. It is AWT-based for maximum compatibility. It uses the x-finder: protocol. It searches for files matching a given pattern. When complete, it prints the number of files.

Example 1

This example illustrates how to start the interactive tool.

run:program:org.jos.gchii.filecount1a.FileCount

Example 2

The following example illustrates how to count the number of .java files in a specific directory tree:

run:program:org.jos.gchii.filecount1a.FileCountProgram
	 ?show=all&directory=/usr/local/src&pattern=*.java

Reference
