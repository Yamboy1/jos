Introduction

The BytecodeProcessFactory class creates an instance of the BytecodeProcess interface with properties from the current application.

Directory property

The user.dir system property is used to set the default value of the directory property. This is the name of the "current" directory, the default directory for opening files in a file subsystem. 
public class _Custom_BytecodeProcess
    implements BytecodeProcess {
  public _Custom_BytecodeProcess() {
    :
    p.setDirectory( System.getProperty( "user.dir", "" ) );
    :
  }
  :
  public String getDirectory() {
    return directory;
  }
  public void setDirectory( String v ) {
    directory = v;
  }
  private String directory;
}
Likely, the directory property is set within the constructor of a platform-specific implementation of BytecodeProcess.
