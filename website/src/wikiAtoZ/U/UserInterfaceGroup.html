This group is intended to focus on the development of user interfaces for JOS.

This working group consists of the following subgroups:

  • JADE - the UI socket
  • ShellGroup - the core plumbing for all interfaces
  • CommandLineGroup - a text interface
  • TraditionalGUIGroup - a traditional windowing system
  • ExperimentalGUIGroup - a bleeding edge component based interface
If you would like a quick introduction, visit UserInterfaceIntroduction.

See UserInterfaceDefinitions.

We are compiling a list of user interface requirements, you can see the current list at UserInterfaceRequirements.

Visit the UserInterfaceNoticeboard to see where we are with the project such as who is doing what and calls for beta testers.

If you have any comments or questions, please type them at UserInterfaceComments.

The mailing list is ui@jos.org you can sign up at http://www.jos.org/mailman/lists/ui
