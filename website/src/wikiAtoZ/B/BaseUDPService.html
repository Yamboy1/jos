SignatureServices

Article contributed by GilbertHerschberger (30 March 2000).

Introduction

BaseUDPService is part of the gchii.signature.service2a package. It can be the starting point for developing your custom UDP service. It can also be the starting point for developing a UDP client.

Date/time example

A date/time service is a good example for UDP. Any client sends a datagram to the date/time service. The date/time service responds with the date/time, according to the server.

Here is an example of the date/time service.

public class DateTimeService
	 <STRONG>extends BaseTCPService</STRONG> {
  private static final int PORT = 1964;
  private static final int SIZE = 100;
  public DateTimeService() {	 
	 super( PORT, SIZE );
  }
  <STRONG>public void onPacket( DatagramPacket v )</STRONG> {
	 String s = "[" + new Date() + "]";
	 byte[] buf = s.getBytes();
	 DatagramPacket packet = new DatagramPacket( buf, buf.length );
	 getService().send( packet );
  }
}

Base class

Here is an example of how the BaseUDPService might be implemented.

public abstract class BaseUDPService
	 extends BaseConsoleProgram
	 <STRONG>implements UDPServiceListener</STRONG> {
  public BaseUDPService() {
	 this( 0, 0 );
  }
  public BaseUDPService( int p, int s ) {
	 setPort( p );
	 setSize( s );
  }
  public int getPort() {
	 return port;
  }
  public void setPort( int v ) {
	 port = v;
  }
  public int getSize() {
	 return size;
  }
  public void setSize( int v ) {
	 size = v;
  }
  public UDPService getService() {
	 return service;
  }
  public synchronized void run() {
	 URI uri = new URI( "x-service:udp:" +
		  "?port=" + port +
		  "&size=" + size );
	 service = (UDPService) uri.getObject();
	 if ( service == null ) {
		return;
	 }
	 service.setListener( this );
	 init();
	 service.run();
  }
  protected void init() {
  }
  private int port;
  private int size;
  private UDPService service;
}
