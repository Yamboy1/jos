Distributed Operating Systems enable a number interconnected computers to operate as a single virtual machine. This is transparent to the user in that they should not be concerned that the program they are running may be running on several different processors across the network. The programer may or may not need to be aware of the interconnections.

Distributed Operating Systems can be broadly classified into:

  • LooselyCoupledSystems where the components that are interconnected are seperate physical machines which are self contained but interact in a distributed environment.
  • TightlyCoupledSystems where the interconnected components are things like processors and memory, i.e. one machine can directly access the memory of another
- IanDavis, 20 Oct 1997
