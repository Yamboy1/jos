Introduction

A data registry is typically a local mechanism. It enables a program to load complex configuration and initialization data from a file, and to save the same to a file.

One such data registry is implemented in the gchii.library.registry1c package. Another data registry is implemented as java.util.Properties.

For more information, see also RegistryPages.
