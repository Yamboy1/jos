ERICProject

Article contributed by GilbertHerschberger (4 January 2001, revised 11 August 2001).

Introduction

The eric tool is part of the ERICProject. It depends heavily upon a Java Runtime Environment (JRE). It uses the Invocation API to start a JNI-compatible virtual machine.

Sometimes, the eric tool can be difficult to install. A JRE can be installed in any directory. Each vendor has their own way of installing a JRE. On Windows, for example, Sun Microsystems updates the Windows registry with the location of its own JRE. Their installation program assumes only one JRE is installed.

Machine code

The installation and configuration program for the eric tool must be written in machine code. If the installation program depends on a JRE, this would be a paradox. The install program would need the javahome property to find a JRE; but, the install program is needed to define the javahome property.

Therefore, a pure machine code program is needed to install the eric tool. Once machine code is selected, it becomes possible to install both the JRE and the eric tool from the same program. It becomes possible to upgrade the JRE and upgrade the eric tool using the "install" program, too.

As a purist, this is a defeat for Java technologies. It should be possible to write an installation program in pure bytecode. Every vendor should have participated in a standard way to register a JRE in Windows. Muliple concurrent JREs are possible and recommended.

sojo tool

The sojo tool is part of the ERIC Project. It is the new front-end for the eric tool. It installs, configures and invokes the eric tool. With the sojo tool, you can
  • install a Java Runtime Environment
  • search for installed JREs
  • launch an application without a shortcut,
  • create an eric shortcut,
  • upgrade the eric tool,
  • launch the Erica utility,
  • launch the UniversalBrowser utility,
  • copy the eric tool for your custom application
  • and much more.
With the introduction of an installation program written in pure machine code, it becomes the only program that needs distribution. The latest JRE can be downloaded with the press of a button. The eric tool can be downloaded with the press of a button. In addition, the Erica utility can be invoked with the press of a button.
  1. Download the sojo utility.
  2. Click the JRE button to download and install a JRE.
  3. Click the ERIC button to download and install the eric tool. This step requires a JRE.
  4. Click the Erica button to invoke the Erica utility or click the Universal Browser button to start the Universal Browser utility. This step requires a JRE and an eric tool.
