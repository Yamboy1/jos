Introduction

Erica is a very cool utility. I am so pleased with the Erica utility distributed with the CjOS Web product. The most recent versions of all programs in CjOS Web are distributed through the Erica utility. This makes my life easier.

The CjOS Web product is installed on the CjOS website at http://cjos.sourceforge.net/cjosweb/ . The Erica utility downloads the package index created by the Deric utility. It determines which package files have changed and downloads them into a cache directory.

This guarantees that my customers are running the latest version of all the programs. This is upgrade-on-demand. Package files are cached on the local machine for maximum runtime performance.

Other applications

You can distribute other Java-based applications using the Erica utility. In fact, you can distribute your own installation-and-upgrade utility using Erica as a platform. Your application or servlet can be distributed directly from your website to your customers through the Internet or your intranet.

The Jar Copier utility is part of CjOS Web. You can install your jar archives on a website and then your customer can run the Jar Copier utility to download them to a local directory. In theory, once you install the CjOS Web product, you never need install any single application again.
