GettingStartedDevelopment; DevelopmentPages

This is intended to act as a guide on how to start a new JOS project. It is assumed you have already registered for a Wiki account, joined the MailingLists, etc. See the main page for how to get started as a JOS developer and the GettingStartedDevelopment page.

To get started:

  1. Read discussions in the appropriate WorkingGroups to check for other people interested in your project area.
  2. Reserve a package name for your project on the Packages page.
  3. Create a Wiki page for your project
  4. Add your project to the ProjectList
  5. Add your project (if desired) to the VolunteersWanted page.
  6. Email the webmaster@jos.org so an announcement will be placed on the jos home page, http://www.jos.org
  7. Write source, and check it into the CVS server and the SourceServer.
  8. Update the NewSourceCode and JosReportCard page.
When you have a product (binaries: beta or better)
  1. Add your product as a link in the appropriate products page with links to the appropriate jars.
  2. Email the webmaster@jos.org so an announcement will be placed on the jos home page, http://www.jos.org
  3. Update the NewSourceCode and JosReportCard page.
