GettingStartedDevelopment; DevelopmentPages

This guide is intended to step you through the process of joining a JOS project. It is assumed you have already registered for a Wiki account, joined the MailingLists, etc. See the main page for how to get started as a JOS developer and the GettingStartedDevelopment page.

  1. Browse the WorkingGroups to see what people are working on.
  2. See if any projects on the ProjectList are interesting.
  3. Get in touch with any of the project leaders to see if you can help.
  4. See if there are VolunteersWanted in an area of your expertise.
