Introduction

The host build of jJOS/decaf enables you to run JOS on top of Linux. This build is often used throughout the development process. The host build helps speed up development.

Test drive

Take JOS for a test drive. Unlike I386Build, this requires a foreign operating system. If you already have Linux installed, you can install JOS quickly and run it as a regular Linux program.

Foreign JDK

Just like the i386 build, this build depends on a foreign JDK. A foreign JDK must be used to compile the JVM-specific classes. Although the JVM-specific classes are platform independent, the build process is written to build the classes on Linux.

Mistakenly, I did not install an off-the-shelf JDK on my Linux box. If I had, this build would have used it to compile JVM-specific classes. Mistakenly, I did not install a copy of classes.zip. At runtime, decaf might have used it.
