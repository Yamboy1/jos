_ Package: org.jos.* _
This is the root package for any "JOS Brand" product, which is portable.

The definition of portable is 100% pure.

Rare exceptions can be made for the org.jos packages, in case a 100% pure version of, say, a library already exists, and it is now desired to change little parts to allow better integration into JOS in a way, the also jos.* packages can benefit from that library without the need to move it into another package. Ask MarkusPeter if you're unsure of what you try to do is what is intended by the above sentence :)

See also jos which is for the JOS "native" code.
