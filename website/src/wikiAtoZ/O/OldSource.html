About this page

This page describes the main JOS source repositories. It is not a list of the available JOS sources. For that see the ProjectList or go to the GettingStartedProducts page. If you are interested in submitting software see HowToSubmitSoftware.

The Source page describes the main JOS source repositories. The ProjectList, GettingStartedProducts and HowToSubmitSoftware pages may or may not be up-to-date.

Two ways

There are 4 main ways to obtain source from JOS projects:

Actually, there are only two main ways to download source from other members of the JOS project.

  1. download with FTP/HTTP from the jos.org or another server, and
  2. download with a CVSClient from SourceForge or another server.

Downloading from Source Server should not be attempted. And downloading via FTP/HTTP is identical regardless of server.

About CVS

1 Check it out of the CVS server. This is by far the preferred method of source storage and retrieval. You can browse CVS source using the CVSWebInterface http://jos.org/cvsweb.cgi

The CVS server on jos.org has been shut down. The JJOS (jJOS/decaf) sister project has moved to the JOS project on SourceForge. The remaining modules have been moved to the CJOSProject on SourceForge.

About Source Server

2 Browse source stored in the SourceServer. This method was initially created to make browsing source online easy. Now that we have the CVSWebInterface, this method of source storage is discouraged. We will probably be removing it in the future. ( IainS 06-11-99 )

Source Server should not be used for uploading or downloading source code. Source code that used to be managed by Source Server is now available elsewhere. For more information, see also DownloadSourceServer.

About FTP/HTTP

3 Download it from the developers site. This typically will be a link on the projects wiki page. Authors often use this method to avoid CVS, or to keep tighter control on access to source (typically when the code is in early development or radical changes are underway).
4 Download it from the JOS webserver. Very few sources are stored on the JOS webserver outside of CVS so this option will rarely be exercised.

Many products have been uploaded via FTP to the jos.org server. Products are found in more than one directory. For example, the JOS Technical Edition is available at http://www.jos.org/redist .

Products are often mirrored on the jos.org server, and should be. No contribution should ever be lost again.
