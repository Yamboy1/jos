Operating System Catalogue

Grasshopper
http://www.gh.cs.su.oz.au/Grasshopper/Papers/GH10/gh10.html
Grasshopper is a operating system that supports OrthogonalPersistence

Tune's List
http://www.eleves.ens.fr:8080/home/rideau/Tunes/Review/OSes.html
Tons of links to every possible operating system desination!

Arizona' List
http://www.cs.arizona.edu/people/bridges/os/full.html
"This pages contains links to past and on going OS projects." - lhsin@pl.jaring.my

WebOS
http://now.cs.berkeley.edu/WebOS/index.html
A project to create a global operating system based around the world wide web. Features include a Web File System that uses the URL namespace, and secure remote execution.

Echidna
http://www.javagroup.org/echidna/
A free multiprocess system in Java - I'd like to talk about agreeing on an interface so the two implementations could be used interchangably.. lukeg@squirrel.net

Elate
http://www.tao.co.uk/taos.htm
A commercial object oriented operating system for embedded systems. They have their own VM. Nice ideas.

Cacao
http://www.complang.tuwien.ac.at/java/cacao/index.html
A 64 bit just-in-time (JIT) compiler for Java for the ALPHA processor.

Grok VM
http://www.envise.com/~pahallam/grok/grok.html
The grok project is an implementation of the Java Virtual Machine for Intel x86 Win32 platforms.

Open Group Research's Test Suite
http://www.gr.osf.org/java/testsuite/html/index.html
Additional testing of Java VM, including performance tests. BillRehm

Utah's Java Computer
http://www.cs.utah.edu/projects/flux/oskit/javaos.html
Another (not so free) Java operating system. It is free for non-commercial use.

JavaOS
http://www.javasoft.com/products/javaos/index.html
JavaSoft's JavaOS(tm) operating system white paper.

Sun's Marketing Collateral For Java OS (tm)

BeOS

Joust
http://www.cs.arizona.edu/scout/joust.html
Project at University of Arizona to create "a small, fast Java OS"

Inferno
http://www.lucent-inferno.com
Lucent's operating system based on their Dis virtual machine. They currently support Limbo, their own programming language and have announced future support for Personal Java.
