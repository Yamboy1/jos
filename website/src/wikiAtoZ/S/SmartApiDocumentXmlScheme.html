Introduction

The document:xml: scheme might simplify your application. It reduces or eliminates references from your application directly to SAX and JAXP code. Because it uses the Input Stream factory, this scheme is expressive, versitile and platform independent.

The general document: scheme is intended to return an instance of org.w3c.dom.Document. The document:xml: scheme returns an instance of Document from an XML document.

Other document schemes are possible. The document:html: scheme, for example, might return an instance of org.w3c.dom.Document from an HTML document.

Plug-in

The XML Document controller is a plug-in for Smart API 4a. It is implemented in the gchii.library.smartxml1a package.

Example 1

Let's say that the example.xml file is stored in the josdemo directory on a local file subsystem. The following identifier is used to load the document: 
document:xml:file:josdemo/example.xml
The following snippet of code illustrates how to load an XML document from a file. 
  public void example() {
    String spec = "document:xml:file:josdemo/example.xml";
    URI uri = new URI( spec );
    org.w3c.dom.Document doc = (Document) uri.getObject();
    if ( doc == null ) {
      _print( uri.getException() );
      return;
    }
    :
  }

Example 2

Let's say that the example.xml file is part of a website on the Internet. The following identifier is used to load the document: 
document:xml:http:jos.sourceforge.net/josdemo/example.xml
The following snippet of code illustrates how to download an XML document from a website. Except for the value of the spec variable, this code is the identical to example 1. 
  public void example() {
    String spec = "document:xml:http://jos.sourceforge.net/josdemo/example.xml";
    URI uri = new URI( spec );
    org.w3c.dom.Document doc = (Document) uri.getObject();
    if ( doc == null ) {
      _print( uri.getException() );
      return;
    }
    :
  }

Example 3

Let's say that the example.xml file is part of a ZIP-compatible archive. The following identifier is used to load the document: 
document:xml:zip:[file:josdemo.zip]/example.xml
Again, only the value of the spec variable changes. 
  public void example() {
    String spec = "document:xml:zip:[file:josdemo.zip]/example.xml";
    URI uri = new URI( spec );
    org.w3c.dom.Document doc = (Document) uri.getObject();
    if ( doc == null ) {
      _print( uri.getException() );
      return;
    }
    :
  }
