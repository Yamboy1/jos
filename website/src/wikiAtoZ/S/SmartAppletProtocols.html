Article contributed by GilbertHerschberger (21 September 1999).

Introduction

A browser, especially an Applet Browser, uses this protocol to load and pass parameters to an applet. An applet factory loads applets and only applets.

Protocol

The following table contains an applet protocol and its well-known interface.

Protocol Interface Description
applet: 
null
java.applet.Applet

This protocol expects an applet name, literally a Java class name. The applet factory is responsible for loading an applet. It does not set the applet's context or call its init() or start() methods.

A client agent must handle a query string. The query string is used to pass parameters to an applet. Since the client agent must provide an appropriate applet context, the factory ignores a query string.
