Article contributed by GilbertHerschberger (21 September 1999).

Introduction

A client agent, especially a Program Browser, uses this protocol to load programs from a given URI. A program factory loads programs and only programs.

Protocol

The following table contains a program protocol and its well-known interfaces.

Protocol Interface Description
run: 
null
Program

This protocol expects a program name, literally a Java class name.

Subprotocols

The following table contains some of the program subprotocols and their well-known interfaces.

Subprotocol Interface Description 
eric:

null
Program

This subprotocol supports a file-spec as well as a query string to set the properties of an ERICShortcut.

 
jose:

null
Program

All four releases of José(tm) are supported. This subprotocol creates a José(tm) application. A José(tm) application does not have a mechanism for program parameters.

 
main:

null
Program

Java applications are supported. A Java application is any class that defines a public static void main( String[] ); method. This subprotocol creates a shell for a Java application. The factory handles program parameter, translating a query string to arguments to main().

 
os:

null
Program

This subprotocol creates a native OS program. The factory handles program parameter, translating a query string to a command line. When run, the result code for a native OS program is returned through program context.

 
program:

null
Program

This subprotocol creates a JOS program. The factory handles program parameter, literally a query string. Since the program context is generic to all client agents, it can be created and populated automatically by the program factory.

A client agent must provide any additional properties.

  • a console or a console program,
  • a terminal for a terminal program, or
  • an IOStream for an IOStream program.

Same protocol in José

This protocol is compatible with the same protocol in Jose. For more information on José-specific subprotocols, see also Run Application inside the José(tm) for Windows(tm) User Guide.
