Introduction

Members have generously contributed their products to the JOS Project. Products include open source and free license software. Products can be downloaded in two ways:

  1. download with FTP/HTTP from the jos.org or another server, and
  2. download with a CVS client from SourceForge or another server.
If you are interested in submitting software see also the HowToSubmitSoftware article on JOS Wiki.

About websites

Many members have their own website. A product distributed by a member may be available on their website. Typically, new releases are announced on the JOS Project homepage. Post your announcement to the jos-general mailing list. Each project should have its own article on JOS Wiki, with a link to the developer's site. Members often use this method because it is easier than CVS, or to keep tighter control on access to source (typically when the code is in early development or radical changes are underway).

About CVS

For many members, CVS is the preferred method of source storage and retrieval. The CVS services on SourceForge and elsewhere enable you to browse source code using a CVS-Web interface.

About Source Server

Source Server should not be used for uploading or downloading source code. Source code that used to be managed by Source Server is now available elsewhere. For more information, see also DownloadSourceServer.
