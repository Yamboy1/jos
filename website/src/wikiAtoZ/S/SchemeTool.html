SmartAPIPages

Article contributed by GilbertHerschberger (8 December 1999).

Introduction

Personally, I would like to have a tool to break off the first scheme from a given uniform resource identifier. In addition, I would like the tool to remove the x- prefix for experimental protocols. When given http://www.jos.org/ or x-http://www.jos.org/, the getScheme() method returns http:. When given an invalid URI, the getScheme() method returns null.

Experimental protocols

Since the SmartAPI is so smart, it can simultaneously deal with many experimental protocols. When you request an object from a smart factory, it does not have to be a system-wide or application-wide smart factory. You can create a localized smart factory, fill it with all kinds of experimental protocols and do whatever you want.

Scheme tool

The scheme tool is one of the first steps in working with a uniform resource identifier (or URI). A scheme is the first group of characters expected in a URI.

public class SchemeTool {
  public static SchemeTool getTool() {
	 if ( tool == null ) {
		tool = new SchemeTool(); // default tool
	 }
	 return tool;
  }
  public static void setTool( SchemeTool v ) {
	 if ( tool == null ) {
		tool = v;
	 }
  }
  private SchemeTool tool = null;
  private static final String CHARACTERS = "abc..z-";
  private static final String EXPERIMENTAL_PREFIX = "x-";
  private static final String ENDOFSCHEME = ":";

  protected SchemeTool() {
  }
  public String getScheme( String v ) {
	 if ( v == null ) {
		return null;
	 }

	 int iMax = v.length();
	 for ( int i = 0; i < iMax; i++ ) {
		String s = ("" + v.charAt( i )).toLowerCase();
		if ( s.equals( ENDOFSCHEME ) ) {
		  if ( v.startsWith( EXPERIMENTAL_PREFIX ) ) {
			 return v.substring( EXPERIMENTAL_PREFIX.length(), i );
		  }
		  return v.substring( 0, i );
		}
		if ( CHARACTERS.indexOf( s ) ) {
		  return null;
		}
	 }

	 return null;
  }
}

Encoding

A scheme has been encoded when a percent sign (%) is used. Following RFC for URL closely, a percent sign is followed by two hexidecimal digits and must be decoded.
