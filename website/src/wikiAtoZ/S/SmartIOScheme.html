Introduction

An I/O stream is capable of both input and output. Examples of an I/O stream include the java.io.RandomAccessFile and java.net.Socket classes.

I/O stream interface

An I/O stream should be reflected in the Java standard class library as an IOStream interface. Instead, an I/O stream interface has been written into the Subprogram API. 
public interface IOStream {
  public abstract InputStream getInputStream()
      throws IOException;
  public abstract OutputStream getOutputStream()
      throws IOException;
}

I/O scheme

The io: scheme is implemented in an I/O stream controller, a plug-in for SmartAPI. It always returns an instance of IOStream stream or null.

Example 1

The following example demostrates how to open a random access file. 
  public void example() {
	 URI uri = new URI( "io:file:example.txt" );
	 IOStream ios = (IOStream) uri.getObject();
	 if ( ios == null ) {
		return;
	 }
	 :
  }

File Adapter

Since java.io.RandomAccessFile does not implement the I/O stream interface, an adapter is needed. This adapter implements both the I/O stream interface and the seek-able interface because a random access file is an I/O stream and seek-able.

Example 2

The following example demostrates how to open a socket. 
  public void example() {
	 URI uri = new URI( "io:socket://localhost:23" );
	 IOStream ios = (IOStream) uri.getObject();
	 if ( ios == null ) {
		return;
	 }
	 :
  }

Socket Adapter

Since java.net.Socket does not implement the I/O stream interface, an adapter is needed. This adapter implements the I/O stream interface because a socket is an I/O stream. Unlike a file adapter, a socket adapter does not implement the seek-able interface because a socket is not seek-able.

Fix this

If someone wants to fix the Java standard class library, here are a few suggestions.
  • alt.io.IOException must extend java.lang.Exception.
  • alt.io.InputStream must be an interface. org.alt.io.BaseInputStream would then be an abstract class which implements alt.io.InputStream.
  • alt.io.OutputStream must be an interface. org.alt.io.BaseOutputStream would then be an abstract class which implements alt.io.OutputStream.
  • alt.io.IOStream must be an interface which extends both the alt.io.InputStream and alt.io.OutputStream interfaces. org.alt.io.BaseIOStream would then be an abstract class which implements alt.io.IOStream.
  • alt.net.Socket should extend org.alt.io.BaseIOStream or implement alt.io.IOStream.
  • alt.io.SeekableStream must be an interface for all seek-able streams.
  • org.alt.io.RandomAccessFile implements alt.io.SeekableStream and should extend org.alt.io.BaseIOStream or implement alt.io.IOStream.
