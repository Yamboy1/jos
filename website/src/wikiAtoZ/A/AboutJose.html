Introduction

The Jose(tm) product is a desktop/browser/shell. It is part of the CjOS Project ( CjOSProject ). It demonstrates one possible implementation of a Subprogram API. For more information, see also the Jose article.

Versions

There are four versions of this product. All versions can run simultaneously in a single virtual machine. Take it for a test drive. Version 1d is the latest.
