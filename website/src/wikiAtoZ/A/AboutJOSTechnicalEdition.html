Introduction

The JOS Technical Edition is part of the JOS Project. It provides an off-line edition of both mailing list archives and JOS Wiki. It provides one-stop shopping for many JOS-related products. It provides syntax-highlighted source code for your favorite HTML browser. Source code is cross-referenced by product and package.

On-line

A limited version of this product is available on-line ( http://cjos.sourceforge.net/redist/ ).

CD-ROM

A full version of this product is available on CD-ROM for approximately US$10.00. (Actual cost depends on shipping charges.) It includes a recent snapshot of JOS Wiki, CVS and other open source products. For more information about ordering, send e-mail ( gchii@mindspring.com ).
