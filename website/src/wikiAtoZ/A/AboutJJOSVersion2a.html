Introduction to version 2a

Like version 1i, this version of JJOS also started with version 1f. Source code in this version is consistently distributed under the JJOS Public License ( JJOSPLLicense ).

The "i386" build boots from both Ethernet and GRUB. The "host" build of this version runs on Linux and compiles on Microsoft Windows.

  • To obtain maximum compatibility with Win32 compilers, C++ source code is stored in .cpp files instead of .cc files.
  • To enable a host build for both Linux and Windows, the C/C++ source code was moved from arch/host/nativecode to arch/host, and from arch/i386/nativecode to arch/i386. The arch/../nativecode directories are used exclusively for compiling on Linux. The arch/host/win32 directory is used exclusively for compiling on Microsoft Windows.
