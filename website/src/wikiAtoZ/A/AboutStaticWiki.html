Article contributed by GilbertHerschberger (6 October 2000, 8 April 2004).

Introduction

JOS Wiki has many articles. More articles are added every month. Static Wiki is an off-line alternative. It is recommended for anyone with a limited Internet connection, for new members of this project, and for anyone doing serious research. Once the static HTML edition has been downloaded or obtained on CD-ROM, an Internet connection is optional. Otherwise, it takes many hours of on-line Internet access to read articles on-line.

Snapshot

Each release is a snapshot. On schedule, a new release is created every three months. While it is most often used off-line, it integrates with the on-line edition for fast access to the up-to-the second changes.

Pages are displayed quickly because all article-to-article links are resolved ahead of time. Use off-the-shelf utilities to perform a full text search. The Text Search utility provided with the Smart API, for example, can find any literal text in under a minute.

The static edition of JOS Wiki does not replace the on-line edition. The on-line edition is required to add and change articles for publication.

Latest release

The latest release is 2a (DownloadStaticWiki).

Release 2a Notes

The integration of this release is more automated. It depends on Ant to download an index of articles from JOS Wiki. It depends on Apache Ant to download pages from JOS Wiki. It depends on Ant to generate a list of new articles. It depends on Apache Ant to build the Static Wiki pages. To build the static HTML pages from source code, Ant depends on the Smart API Task ( AboutSmartApiTask ). In turn, Smart API Task depends on the latest release of WebSource.
