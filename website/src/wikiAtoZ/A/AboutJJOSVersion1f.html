Introduction to version 1f

This version does not exactly work. There is no corresponding binary edition.

Since 2a is a slight improvement in version 1f and 2a works with the jjos.zip from version 1e, this version could be made to work with a few simple patches.

In fact, I am planning to apply these patches and commit them to JJOS CVS.

Module

This version is stored in the jjos1f module on JJOS CVS on SourceForge.

Source

The "official" source code for release 6 is found in the following archives:
  • http://www.jos.org/redist/mirror/jos1f/jos1f-module-JJOS-6.tgz
JJOS CVS on SourceForge provides another copy of the source code.
  • http://jjos.sourceforge.net/
Does the source code in the "official" archive match the source code available through JJOS CVS?

Java source code

There is a problem with the Java source code for this version. It won't compile as written. It won't run even with the following patches:
  • In Java, there is no counter.java source code. The counter class is missing. I created a stub class--just enough to let the TestSuite class compile.
  • In TestSuite.java, jos.system.keyboard cannot be imported. I changed this to jos.platform.keyboard--just to let it compile.
  • Again, in TestSuite.java, jos.platform.device.StandardVga cannot be imported. I changed this to jos.platform.device.standardVGA.
