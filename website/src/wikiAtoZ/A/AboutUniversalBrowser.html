Introduction

The Universal Browser utility is part of the Smart API subproject. It is an AWT-based utility for maximum compatibility. It demonstrates the power of the Smart API. Press the Run button and almost any URI typed into the location field will be turned into an object.

Starting

To start the Universal Browser utility, type in the following command:

java -jar universalbrowser.jar

The UniversalBrowser class is the main class for this utility. It is found in the org.jos.gchii.browser1b package.

About combo edition

The runtime edition for this product can be assembled by combining the following archives from other runtime editions:

  • smartapi1a.jar
  • smartapi1b.jar
  • smartapi2a.jar
  • smartapiutil1.jar

The source code for this edition is distributed separately.

  • Smart API 1a - Release 4
  • Smart API 1b - Release 4
  • Smart API 2a - Release 15
