BulkMake; DownloadBulkMake; CjOSPages

Article contributed by GilbertHerschberger (2 October 2000, revised 4 August 2002).

Introduction

BulkMake(tm) is maintained by the CjOS Project. This Java-oriented product copies, compiles and archives your Java source code. It archives your compiled Java code. It does all this without a project file or make file.

Recommended usage

A large project, such as the CjOS Project itself, has hundreds of packages. This product has been specifically designed for medium sized projects, with 15 or more packages. To be sure, it might be overkill for small projects with fewer packages.

This product is as platform-independent as possible. It has been designed for multiple-platform projects. Programmers sometimes do not include a project or make file that goes with your compiler, platform or IDE.

Latest version

The latest version of this product is version 2d. It is available in a multiple platform version and a version specifically designed for Microsoft Windows(tm). For more information, see also DownloadBulkMake .

History

This product has been used in a production environment for more than a year. It has also been used to distribute JOS-related products inside the JOS Technical Edition. For more information, see also AboutJOSTechnicalEdition .
