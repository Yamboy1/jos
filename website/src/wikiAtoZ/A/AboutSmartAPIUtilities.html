Introduction

Many utility programs have been based upon the Smart API. Utility programs are AWT-based for maximum compatibility across Java platforms. Eventually, the Smart API will be distributed separately from its utilities.

Utilities

This product contains the following utilties:
  • AppletBrowser
  • Connection Browser
  • Desktop Browser
  • FileCount
  • FileNameLength
  • FindUtility
  • FinderBrowser
  • InstanceOf
  • ObjectBrowser
  • PersonalWiki
  • PlainTextBrowser
  • Program Browser
  • Properties Browser
  • Smart Accepts
  • Smart List
  • Text Compare
  • TextSearch
  • Token Browser
  • UniversalBrowser
