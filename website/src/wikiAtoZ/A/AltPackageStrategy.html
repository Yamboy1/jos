AltPackagePages

Article contributed by GilbertHerschberger (22 January 2001).

Introduction

The following strategy has been selected for building some of the alternative packages.

  • Interfaces and exceptions should be "ported" first.

Interfaces

All interfaces are similar. Start a new interface with the source code from a similar one.

  • AltLangRunnable

Exceptions

Many exceptions are similar. Start a new exception with the source code from a similar one. Alternative exceptions should implement the alt.lang.Throwable interface, not java.lang.Throwable.

About Jasmin

The Jasmin programming language enables us to create an implementation of core Java packages without regard for the version of a JDK. When cross-compiling Java 2 with a Java 0 compiler, the poor design of the standard javac tool is inadequate.

Preprocessor

An in-line preprocessor for Jasmin can be developed, and should be. Jasmin reads from any implementation of the java.io.InputStream class. An in-line preprocessor would use (read from) an InputStream and implement an InputStream. It would resolves compiler directives, such as these:

#ifdef ALT
.interface public alt/lang/Runnable
#else
.interface public java/lang/Runnable
#endif

Values are defined for the preprocessor and alter the text passed to Jasmin. This enables single-source implementations of multiple packages.
