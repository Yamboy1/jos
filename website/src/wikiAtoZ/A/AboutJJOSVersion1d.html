Article contributed by GilbertHerschberger (21 November 2001).

Introduction

The jjos-common.zip archive contains the class files from version 1d of JJOS. Unlike later versions, the arch/host/bytecode and arch/i386/bytecode directories contain a jjos-common.zip archive, but no Java source code. This is as it should be.

New binary edition

A new binary edition of JJOS 1d is coming. (It's here!) Changes in the jjos1d module will be committed to CVS and then a corresponding archive will be uploaded to our mirror directory.

make install-grub

For the i386 build only, the install-grub target has been added to the Makefile. It saves jjos-grub as /josboot/jjos1d-g and jjos.zip as /josboot/jjos1d.zip.

make install-nbi

For the i386 build only, the install-nbi target has been added to the Makefile. It saves jjos1d-nbi as /tftpboot/jjos1d-nbi.

About GRUB

The GRUB-compatible edition is not yet available. Some of the source code is missing. There is no source, although the make script refers to the corresponding assembly language program.
