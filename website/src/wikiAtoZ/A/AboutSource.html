Introduction

Source code is very important to any open source project. Because there are so many JOS-related projects, there is more than one way to obtain JOS-related source code. This is an overview of where you can find source code and how you can obtain it. This article is organized by catagory.

SourceForge

SourceForge is the current host of many JOS-related products. Among developers, CVS is a popular way to distribute source code. The SourceForge provide a free CVS service for open source projects. To download source code for a product, see also individual project pages.

JOS Technical Edition

The JOS Technical Edition ( AboutJOSTechnicalEdition ) is available both on-line and on CD-R. A limited version is found on-line at http://cjos.sourceforge.net/redist/ . Its mirror directory provides one-stop shopping for many product archives.

Source Server

This is an obsolete method of obtaining source code. Source code stored inside Source Server at jos.org has been committed to CJOS CVS ( CJOSProject ) on SourceForge.
