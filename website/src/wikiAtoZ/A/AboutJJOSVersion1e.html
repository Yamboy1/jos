Introduction to version 1e

This version works. It boots from GRUB. It responds to keystrokes and switches between 7 virtual consoles. In this version, console 7 displays (eventually) a bit-mapped JOS logo.

The binary edition of JJOS 1e is available from a website on the Internet. Where is the corresponding source code? What is required to build the binary edition from its source code? Corrado Santoro was kind enough to build a binary edition for us. Until a few months ago, I couldn't build any binary edition using my g++ compiler.

Module

This version is stored in the jjos1e module on JJOS CVS on SourceForge.

Source

The "official" source code is found in the following archives:
  • http://www.jos.org/redist/mirror/jos1e/josgrub-1e.tar.gz
JJOS CVS on SourceForge (JJOSCVSOnSourceForge) provides another copy of the source code.
