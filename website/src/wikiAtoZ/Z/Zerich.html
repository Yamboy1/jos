Introduction

Zerich(tm) is part of the ERICProject. It is written in C/C++. It provides a platform-independent SharedLibrary class for Linux(r) and Windows(tm).

The zerich shared library will be statically linked into an eric tool for Linux(r). The external subroutines and classes will be defined in zerich.h, in its own include directory.
