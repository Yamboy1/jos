WebObjects is a web-application development environment created by NeXT and currently owned by Apple. It works with the OpenStep environment and is deployable on NT and several UNIX flavors. OpenStep and WebObjects are a huge part of the core technologies included in Apple's next-generation Rhapsody OS that will allow for cross-platform application development on Rhapsody. For more info on WebObjects, go see the Apple Enterprise web site.
