Introduction

If possible, you should join an existing sister project already in progress. Why? Because an existing sister project has source code you can download. You can get started more quickly.

Mix in good company. Join a sister project to interact closely with other experienced programmers. Some have been working on a Java-based operating system for four years or more. Get advice that works.

Our sister projects are in need of writers, testers, coders and other volunteers. If you're ambitious, join more than one. Otherwise, you might consider WhyStartSisterProject.

Join the JJOS Project

This team is currently working on version 2a. They are working to document different versions and compare them. What are their strengths and weaknesses? If you'd like to join this sister project, send us a message on the jos-general mailing list.
  • License: JJOSPLLicense
  • Development Platform: Linux and Microsoft Windows
  • Programming Language: C/C++
  • Target Hardware: IBM PC compatible, Intel 80386+
  • Tools: JDK 1.1.x, make, BulkMake
  • Builds: Etherboot, GRUB, Linux, Windows
  • Project Owner: ToddMiller

Join the teaseme Project

This team has accomplished much. They have been able to build a single-source virtual machine boot-able with RJK. Starting with the kissme virtual machine, they have created a Linux kernel module and, of course, a host build. If you'd like to join this sister project, send us a message on the jos-general mailing list.
  • License: GPL
  • Development Platform: Linux
  • Programming Language: C
  • Target Hardware: IBM PC compatible, Intel 80386+
  • Tools: JDK 1.1.x, makeme
  • Builds: RJK, Linux kernel module, Linux
  • Project Owner: JohnLeuner

Join the CjOS Project

This team is working on a virtual machine as a series of common virtual machine components, part of a "VM Kit". And, there is a lot of operating system utilities written in the Java programming language in anticipation of a working Java-based operating system. CJOSCvsOnSourceForge is a source code repository. It contains contributions from many members of the JOS Project, such as the eric tool (C++), the Data Registry API and the Smart API (Java). If you'd like to join this sister project, send us a message on the jos-general mailing list.
  • License: GPL
  • Development Platform: Linux and Microsoft Windows
  • Programming Language: C/C++, Java, Jasmin
  • Target Hardware: IBM PC compatible, Intel 80486+
  • Tools: JDK 1.0.x, JDK 1.1.x, BulkMake
  • Builds: Linux, Windows
  • Project Owner: GilbertHerschberger
