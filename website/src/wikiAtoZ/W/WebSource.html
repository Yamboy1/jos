DownloadWebSource; DownloadUtilities

Article contributed by GilbertHerschberger (13 July 2000).

Introduction

WebSource(tm) is a programming utility. It is part of the JOS Project. It is an open-source and free license software. It generates static HTML pages from source code. Currently, it provides syntax-highlighting for the Java programming language.

WebSource(tm) is available for immediate download ( DownloadWebSource ).

Example

WebSource(tm) generated all of the source code pages for the JOS Techical Edition. Here is an example ( http://www.jos.org/redist/websource/ ) of the pages WebSource(tm) generated for its own source code.

Features

Here are a few of its features.
  • Inter-class links. For the Java programming language, this utility provides automatic links from package to class, from class to package and from one class to another.
  • Unfinished source code. This utility works with both finished and unfinished source code. It does not pretend to understand the syntax of the Java programming language. It uses a simple line-by-line translation.
  • Configureable. Configuration includes the tag line written at the top of each generated page, custom keywords, custom color selection and package alias. Tags lines are important to link source code to its product documentation. Custom keywords are important to adapt quickly to changes in the Java programming language. Custom color selection is important for integrating generated pages into your own website. Package alias is important for generating static HTML pages for a CD-ROM.
  • Extendable. The model is simple and easily extendable. The JavaSourceProgram class can be extended or replaced.
