8 August 2000

  • The HTML pages for JoZilla has been upgraded to fit on a CD-ROM. Some of the page names were too long. Using an important feature of WebSource, jz-1 is an alias for org.jozilla.

    • View source

  • The HTML pages for BulkMake have been updated to combine both version 2a and 2b. The DownloadBulkMake link has been added to this article.

    • View source

  • The index of All Packages has been updated. It includes JoZilla and BulkMake.

GRUB 0.5.95 has been added to the full version. Both the source distribution (grub-0.5.95.tar.gz) and binary distribution (grub-boot-0.5.95.image) are found in the download directory. -- GilbertHerschberger

2 August 2000

  • There has been a change in CVS. ToddMiller has moved the JJOS module from JOS to SourceForge. A snapshot taken on 1 August 2000 is available for immediate download.

  • The Glossary has been updated. Both new and lost articles have been added.

  • JoZilla (JosBrowser) has been added to this distribution. View source

  • Broken links have been fixed in the limited on-line version of JOS Technical Edition - Release 1g. The full version is now available on CD-ROM for US$10.

  • Coming soon: A snapshot of JOS Wiki was taken on 1 August 2000. Both new articles and LostArticles have been added. The archive has not yet been uploaded. Glossary pages from A thru Z, LostArticles, and this page need to be downloaded again.

Links to download pages are provided at the top of this page. Additional download pages are needed for BCNI, AltBridge and others. -- GilbertHerschberger

31 July 2000

It is time for the next edition of JOS Technical Edition. The focus of this edition is the lost articles on JOS Wiki. The Glossary will be updated. The JoZilla project has been rediscovered and it will be added to the product list. Links for the on-line version will be cleaned up.

There has been a change in CVS. ToddMiller has moved the JJOS module from JOS to SourceForge. A new snapshot of the JJOS module will be available soon. -- GilbertHerschberger
