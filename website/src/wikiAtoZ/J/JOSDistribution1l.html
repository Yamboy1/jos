AboutJOSTechnicalEdition

30 August 2001

The mailing list archives on the www.jos.org server contain more than 4MB of messages. These messages will be part of the JOS Technical Edition, starting with version 1l.

A link to the mirror directories will be more prominent in this version. A Mirror link will be provided on the homepage, in addition to a link on the Directories on CD-ROM page.

With Static Wiki and now Static Mailman, both the wiki database and mailing list archives are available off-line. Much can be learned without a high-speed Internet connection.

-- GilbertHerschberger
