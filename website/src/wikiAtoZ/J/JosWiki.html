JosWiki is a clone of WardCunningham 's WikiWikiWeb.

Perl Sourcecode and example templates are available via anonymous ftp from ftp://ftp.jos.org/pub/wiki.zip and ftp://ftp.jos.org/pub/exampletemplates.zip

Note that there's no documentation and that it's rather hard to setup. Help make it easier and join the HowToInstallJosWiki group.

You may look at HowToInstallJosWiki and JosWikiAddOns for more information on how to setup your own JosWiki !

Notice: The JOSWiki is now a clone of the TWiki software

(TWiki was cloned from the original JosWiki which is a WikiWiki clone...). Please see http://TWiki.SourceForge.net/ for more information and download/install instructions.

Happy Twiki'ing.
