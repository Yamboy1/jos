AndrewReutter, who just joined the JoZilla Project Team, has just coined a new word for a subapplication that is designed to run inside the JoZilla Browser: Browselet.

Here are some Browselets examples:

  • HTML Renderes
  • WebCrawlers
  • Visual Web Site Mappers

A Virtual Browser Window Manager is an idea that would would allow JoZilla to maintain a tiny visual object that represents a new Browser Window which was "opened" by a target="_new" attribute inside an Anchor Tag. This will give the user a chance to decide whether or not to allow new browsers to open up. It has been noted by OmegaBaphomet that while surfing "hostile" Adult Web pages, so many pages open up for "advertising" that the pleasure of the experience is reduced. With this feature enabled one can just surf at Will and snicker as the new virtual browser windows are merely visually displayed. Theoretically the virtual browser windows can be deleted with a single mouse click on a "delete all virtual browser windows" widjet.
