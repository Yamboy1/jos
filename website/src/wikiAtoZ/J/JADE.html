JADE

JADE stands for the JOS Application and Desktop Environment.
JADE is an application of Java, requiring JOSystem, that allows a UI to be "plugged" into JOS creating a proxy between the OS and the user.

Requirements:

NOSHADE WIDTH="50%">
  • Run on JOS with a minimum set of drivers/extensions (even though more will be needed for stuff like a GUI)
  • Take the burden of user interaction off the OS while offering max code reuse to the pluggable UI's
