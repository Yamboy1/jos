Introduction

A package implemented in the Java programming language can be converted to a corresponding class library in C++. A package is the equivalent of a class library in C++.

Each package is a shared library with a corresponding header file. When generating C++, Java should be converted at the package level. A package-at-a-time approach enables the generator to build one header file per package, not one header file per class.

Once compiled with C++, each package becomes a shared library. As a shared library, it implements the mechanics of the SharedLibrary class from the ERICProject. The getMethod() method expects a method name in the form of class::method.

Interface

An interface in Java is equivalent to a pure virtual class in C++. When a java->cpp translator creates C++ code, it only creates a header file. A pure virtual class has no implementation. It is only an interface.

A namespace of Java might be mapped to a namespace in C++. A package name is mapped to a C++ class.

package josdemo;

public interface Demo {
  public abstract void example();
}

class josdemo {

  class Demo {
	 public:
		<STRONG>virtual</STRONG> void example() <STRONG>= 0</STRONG>;

	 protected:

	 private:

  };
};

Class

A class in Java is equivalent to a class in C++. When a java->cpp translator creates C++ code, it creates a header file. A C++ programmer must fill out the corresponding implementation.

package josdemo;

public class Demo {
  public void example() {
	 String s = "Hello, World!";
	 System.out.println( s );
  }
}

class josdemo {

  class Demo : public javalang::Object {
	 public:
		<STRONG>virtual</STRONG> void example();

	 protected:

	 private:

  };
};

void josdemo::Demo::example() {
  cout << "Hello, World!" << endl;
}

Prequisites

In order to compile the C++ code generated from Java package, a C++ compiler must have classes that are pre-defined and available.

  • javalang::Object

    This class serves as a base class to all Java-compatible classes.

  • javalang::Class

    This class is returned by javalang::Object.getClass(). In C++, this is a special class. It cannot be constructed without using a factory method.

    class javalang {
  class Class : public Object {
	 public:

	 protected:

	 private:
		Class();
		Class( const Class &other );
		const Class &operator==( const Class &other );
  };
};

  • javalang::String

    This class is used as the primary string class.
