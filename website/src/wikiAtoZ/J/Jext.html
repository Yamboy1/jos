Application: Jext

Type: Stand-Alone/Application

Package: org.jos.app.editor

Objective: To produce a fully featured text editor mainly designed for Java, C, C++, Perl and HTML/XML users, but which can be very efficient for others, too.

Visite the website: http://www.jext.org
Join the mailing list: http://mail1.sourceforge.net/mailman/listinfo/jext-devel

TESTERS NEEDED - SEND ME A MAIL !! Until I've uploaded the sources on the CVS servers, you can download Jext at: http://www.jext.org. Note: dev releases are more interesting.

This application contains the following features inherently:

  • Dawn internal scripting in version 1.1 (Java support)
  • Project manager
  • Remote telnet server
  • Multi document edition
  • Colorizing file filters
  • Files finder
  • Gutter
  • Key shortcuts edition
  • Type Saver (press ESC on a word defined as an abrev to expand it)
  • Console
  • Plugins support
  • Scripts support
  • Syntax colorization for Java, Perl and HTML/JavaScript
  • Open zip files and extract any file from'em
  • Source beautifier
  • Auto check version from Web
  • Internal and external properties based on XML
  • Customizable insert menu based on XML
  • GUI entirely built from XML files
  • Mail your files from Jext !
  • Open files from the Web !
  • Unlimited Undo/Redo
  • Basic editing - cut, copy, paste and select all
  • Shortcut keys, Drop down menus, Floatable Toolbar
  • Auto Indent
  • Selectable Font, size, etc and Tab width
  • File open, save, auto-save, close, exit
  • Zip File As and Save As
  • Find and Replace (using gnu's regexp package)
  • Case conversion [UPPERCASE, lowercase]
  • Insert Date/Time
  • Methods Seeker (for Java sources editing)
  • Class browser
  • HTML Indenter
  • Sort lines
