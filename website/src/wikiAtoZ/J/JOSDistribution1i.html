6 January 2001

Changes in the CVS services have caused corresponding changes in the JOS Technical Edition. A CVS service is no longer running on the jos.org server. Only the JJOS module has been moved from jos.org to cvs.jos.sourceforge.net.

Meanwhile, the CJOS Technical Edition is available on a separate CVS service and separate CD-R. It is a separate product. It includes the jos and projects modules. It includes three Source Server modules. It includes the full source code to the Universal Browser. The jos and projects modules have been moved from jos.org to cvs.cjos.sourceforge.net. -- GilbertHerschberger

23 December 2000

Some of the source code introduced through the JOS Project is in need of simple repair. For example, some Java program files simply have the wrong package name. Other Java program files have been moved and renamed. Because an original author can no longer maintain it, the source cannot yet be repaired in an official way. Unfortunately, each member must fix the source for themselves. This may be the theme of the next release.

When an open source project grows beyond the scope of its original developer, it should be maintained by someone else. If no individual volunteers, the JOS Project as a group becomes responsible for contributed code.

A Concurrent Versioning System (CVS) service might be a good mechanism for a group effort to maintain the source code. SourceForge provides a free CVS service for open source and free license software. Use an existing project or start your own.

The CJOS Project is found in CVS on SourceForge. It includes a lot of source code from the JOS Project, but not all. It includes source code contributed to Source Server on jos.org. It includes Universal Browser and all its related packages. It includes most, if not all, of the source code I've personally contributed. But, it does not yet include products written by some authors, even those products available in the JOS Technical Edition.

In the future, the CJOS Project will be limited to code licensed under the GNU Public License (GPL) or GNU Lesser Public License (LGPL). Any code maintained through the CJOS Project must be licensed (or re-licensed) as GPL/LGPL. -- GilbertHerschberger
