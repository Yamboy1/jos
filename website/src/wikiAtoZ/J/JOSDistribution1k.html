25 June 2001

This distribution starts with the creation of Static Wiki. Since JOS Wiki has been moved from www.metamech.com to wiki.jos.org and redeployed from TWiki to sfWiki, the creation of Static Wiki for the JOS Project must use a different suite of programs. These programs are written in the Java programming language.
  • A list of articles has been downloaded from the new JOS Wiki. Fortunately, the changes program of sfWiki software provides a complete article index.
  • A snapshot of JOS Wiki was created this afternoon (Monday, 25 June 2001) at 1:00pm EDT. The snapshot saves articles as is--without converting them to usable cross-linked static HTML pages.
  • The new Static Wiki will maintain the look-and-feel of the new JOS Wiki. The new template contains fewer links to wiki articles. More work is required to convert all links in the snapshot into proper links for Static Wiki.
  • For more information on Static Wiki, see also the AboutStaticWiki article.
-- GilbertHerschberger
