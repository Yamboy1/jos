AboutJOSTechnicalEdition

12 February 2002

  • Added the Jose product ( AboutJose ) to the JOS Technical Edition. The jose1d directory was created for its source code formatted in HTML.
  • Moved the on-line version of the JOS Technical Edition from jos.org to cjos.sourceforge.net. For more information, see also SpecialLink article.
-- GilbertHerschberger

2 January 2002

  • Downloaded another snapshot of JOS Wiki, after updating the Glossary, creating a few new articles and editing a few others.
-- GilbertHerschberger

1 January 2002

  • Upgraded Wiki Changes program from version 1a to 1b. The new version preserves the original article names.
  • Added the Khimera product ( AboutKhimera ) to the JOS Technical Edition. The khimera1d directory was created for its source code formatted in HTML.
  • Downloaded a snapshot of JOS Wiki to create the next version of Static Wiki ( AboutStaticWiki ).
  • Created preview of Static Wiki 1m. Unfortunately, links at the top of every page point to http://wiki.jos.org/ rather than http://jos.sourceforge.net/ . Unfortunately, the 'recent changes' link points to NewArticles1k instead of NewArticles1m.
-- GilbertHerschberger

26 December 2001

  • Machine names. jos.sourceforge.net and wiki.jos.org used to be the same machine.
    • For more information, see also JOSProjectAndDNS .
  • About Static Wiki. A new Static Wiki shapshot will be downloaded from jos.sourceforge.net instead of wiki.jos.org. It will include this and other articles about the JOS Technical Edition.
    • For more information, see also AboutStaticWiki .
  • Source code. Browse the Java source code using your favorite HTML browser. The migration initiative for the CjOS Project has touched a lot of source code. This initiative has moved Java packages from the com.realignsoft namespace to the gchii or org.jos namespace.
    • For more information, see also CjOSAndJOSTechnicalEdition .
-- GilbertHerschberger
