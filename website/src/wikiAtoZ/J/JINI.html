JINI Information

Jini stands for "Jini Is Not Initials".

Sun is shipping the Jini source code under the "Sun Community Source License". I (MattAlbrecht) communicated with some Sun officials, and they said that JOS could use Jini with the Commercial Use arrangement. This requires us to sign a commercial use community source license with Sun, which requires of us:

  • JOS must pass the (freely available) Jini Technology Compatibility Kit
  • Sign a separate Jini Compatibility logo license.
Normally, this would require a "minimal" fee, but since we're a non-prophet org ( :-) ), we don't have to pay.

-- MattAlbrecht - 17 Nov 1999
