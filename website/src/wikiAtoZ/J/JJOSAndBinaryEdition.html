Introduction

JJOS is one prototype of a Java-based operating system. It is open source and free license. It is part of the JJOSProject.

For the record, Corrado Santoro generously volunteered to build the binary editions and maintain the download page. His JOS Binary Edition page was the first to provide links to archive'd binary editions of JJOS. Binary editions enable anyone to take JJOS for a test drive--even without a full set of development tools.

For more than a year, these archives have also been made available throught the JOS Technical Edition ( http://cjos.sourceforge.net/redist/ ). These archives are not lost; they are still available today.

The JJOS Project Homepage is the official source of binary and source editions today. Click on the "mirror" link for a list of archives.

Plan

Improvements to the binary editions are planned. Here is part of the current plan.
  • Each binary edition should include information about how to install and run it.
  • Each binary edition should include license information. JJOS is distributed under the JJOS Public License ( JJOSPLLicense ).
  • Each binary edition should include a corresponding make file to install the binary edition for Etherboot or GRUB.
  • GRUB-compatible executables, GRUB-compatible archives and Etherboot-compatible executables will be uniquely named for each version of JJOS, so that an end-user can install every version at the same time.
  • The binary editions will be re-distributed without recompiling, if possible.
  • The names of the distributed archives will be more consistent, making room for future versions.
  • Etherboot-compatible executables should be "installed" into the /tftpboot directory. GRUB-compatible executables should be "installed" into the /josboot directory.
