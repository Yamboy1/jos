Process

The JOS Project intends to make high quality source code available to the global community. Source code is a critical component of any open source project. This project is part of an effort to recapture the working source code already contributed to the JOS Project over the last few years.

On a class-by-class basis, source code from jos1c-srcserv.tar.Z will be integrated into future distributions of jos1d-srcserv. For the current distribution of source server, see also JOSDistribution1d.
