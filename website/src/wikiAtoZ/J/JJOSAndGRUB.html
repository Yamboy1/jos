JJOSPages; InstallPages

Article contributed by GilbertHerschberger (17 May 2001).

Introduction

The I386Build build of JJOS is compatible with Etherboot and GRUB. The GRUB version of JJOS consists of the jjos-grub kernel and jjos.zip module. When jJOS and decaf "libraries" are statically linked together, this is the current jjos-grub kernel. The jjos.zip module must contains all bytecode required by your application because JJOS does not yet support an external file subsystem.

Summary

Here is a summary of commands entered at the GRUB prompt: 
grub> root=(hd0,0)
grub> kernel=/josboot/jjos-grub jos/i386/i386init
grub> module=/josboot/jjos.zip
grub> boot

Procedure

The following procedure illustrates how to boot JJOS with GRUB.
  • Prerequisite: You'll need a copy of the jjos-grub and jjos.zip files in the /josboot directory. A binary edition of JJOS is available as an archive. With GNU/Linux, you can make these files from source code with the cd arch/i386/nativecode; make install-grub command.
  • Prerequisite: You'll need a GRUB Boot Diskette. To create this with Microsoft Windows or GNU/Linux, use the Boot Disk utility (InstallWithBootDiskApp).
  1. Insert the GRUB Boot Diskette into your first floppy drive so that, upon restart, your machine will boot from floppy instead of a hard drive.
  2. Press your computer's reset button to restart your computer. As a result, your computer will display the opening GRUB messages and then a GRUB prompt (grub>).
  3. At the prompt, type the following command and press the Enter key. 
    grub> root=(hd0,0)
    An error message may indicate that you need to specify a different drive or partition. Within the (hd0,0) parameter, the first number is the drive and the second number is the partition.
  4. At the next prompt, type the following command and press the Enter key. 
    grub> kernel=/josboot/jjos-grub jos/i386/i386init
    The program should confirm a boot image in ELF format. An error message may indicate that the file does not exist. Check the /josboot directory for the jjos-grub file. Otherwise, your /josboot directory may be on a different drive or partition.
  5. At the next prompt, type the following command and press the Enter key. 
    grub> module=/josboot/jjos.zip
    Likewise, the program should confirm a module file. Check the /josboot directory for the jjos.zip file.
  6. At the next prompt, type the following command and press the Enter key. 
    grub> boot
    The program will launch the JJOS software.
