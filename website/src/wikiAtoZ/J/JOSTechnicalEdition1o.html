AboutJOSTechnicalEdition

Tuesday, 6 August 2002

  • Started work on new release of Static Wiki.
  • Release 1 of BulkMake 2d is now available for download. This version includes new options for the package archive program. It includes a built-in Clean tool, which uses the package cleaner program to remove class files. See also DownloadBulkMake2d .
  • Release 1 of JOS Static Wiki 1o is now available. See also DownloadStaticWiki .
-- GilbertHerschberger

Friday, 19 July 2002

  • Created new title graphic for the JOS Technical Edition so that the title for version 1o says "1o".
  • Revised the Static Wiki 3b program to substitute local links when possible. The mailing list archives are available locally, for example, so articles in the static wiki should refer to them instead of the archives on the non-existant www.jos.org server. Links should point to the local download and mirror/* directories instead of pointing to directories on the Internet. "View source" links should point to the index.html page in a WebSource directory instead of displaying a directory listing.
  • Release 2 of Smart API 4a is now available ( DownloadSmartAPI4a ). This version does not completely replace versions 1a, 1b or 2a. Version 4a depends upon 2a. In turn, version 2a depends upon 1a and 1b.
  • Release 1 of Static Wiki 1o is now available. It is available as both a ZIP-compatible (.zip) archive and a compressed tar (.tgz) file.
-- GilbertHerschberger

16 July 2002

  • Created NewArticles1o article on JOS Wiki. It contains links to new articles written in the last six months or so.
  • The title graphics for the JOS Technical Edition have not yet been updated. While it still says version 1m, it is really version 1o.
  • As requested (Thanks, Andy!) the on-line version JOS Technical Edition is now available in one big downloadable file. Look for the "joste1o-1-online.tgz" file in the http://cjos.sourceforge.net/mirror/ directory. It is 167MB.
  • I have already downloaded a snapshot of JOS Wiki articles for inclusion in the next version of Static Wiki.
  • Updated the org.jos.smartapi4a package in Java for Smart API 4a. It seems that getController() and setController() methods were mistakenly omitted from the URI class; it does not match the design.
  • Upgraded the Package Archive product from 2c to 2d. The compression property has been added. The compression property is true by default. When true, a jar archive (.jar) file is created with compression and a manifest file. Otherwise, a ZIP-compatible archive (.zip) is created without compression and without a manifest file.
  • Updated the gchii.tools.bulkmake2d package in Java for BulkMake 2d. Added the compression property to the archive tool. Switched Package Archive from version 2c to 2d. (And it is only coincidence that BulkMake and Package Archive are both version 2d.)
  • Upgraded the Static Wiki software from version 3a to 3b. Version 3b corresponds to version 1o of the JOS Technical Edition. Within static wiki pages, automatically switched from www.jos.org/redist to cjos.sourceforge.net/redist.
-- GilbertHerschberger

9 July 2002

The limited on-line version ( http://cjos.sourceforge.net/redist/ ) of the JOS Technical Edition has been updated to version 1o. This version includes the following changes.
  • Updated the BulkMake product directory. Version 1d is the latest version.
  • Added the Master Compiler product directory. Version 2a is the latest version.
  • Updated the Smart API 1b product directory. Release 7 is the latest release.
  • Updated the Smart API 4a product directory. Release 1 is the latest release.
  • Updated the Smart API Utilities product directory. Release 5 is the latest release.
  • Updated the doc, src and stub directories. Updated the All Packages Index page.
  • Updated the index.html file, but have not yet created the new title graphic.
This is the first release of version 1o.
  • A distribution of BulkMake 2d is planned.
  • An update to the glossary for JOS Wiki is planned.
  • A new version of the Static Wiki product is planned. See also AboutStaticWiki .
-- GilbertHerschberger
