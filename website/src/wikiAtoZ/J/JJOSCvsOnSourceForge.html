Introduction

JJOS CVS on SourceForge refers to a part of SourceForge reserved for files from the JJOS Project. These files are available through a well-known CVS prototol. This is an important part of the open source JJOS Project.

What is it? The folks at SourceForge were kind enough to let us open source programmers use their equipment and services. The JJOSProject is an open source project. The CVSService enables contributors to upload their files. A CVSClient enables others to download these files from a server. Here, a CVS service runs on servers at SourceForge and a CVS client runs on a machine connected to the Internet.

Oh no! The java0 module is not part of the JJOSProject. I imported it by mistake. It should be removed immediately.
