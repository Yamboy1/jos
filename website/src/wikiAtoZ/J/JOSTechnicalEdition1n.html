AboutJOSTechnicalEdition

4 April 2002

Version 1n of JOS Technical Edition has been started.
  • This is the first version to include version 4a, the latest version of the Smart API.
  • This is the first version to include version 2a of the Master Compiler from the CjOS Project.
  • The binary edition of the eric tool is planned.
  • The distribution of ERIC for Windows is planned.
  • The binary edition of the kick tool is planned.
  • An update of the JOS Wiki glossary is planned.
  • Version 1n of StaticWiki is planned.
-- GilbertHerschberger
