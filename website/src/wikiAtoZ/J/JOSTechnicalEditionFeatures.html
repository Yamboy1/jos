AboutJOSTechnicalEdition

Article contributed by GilbertHerschberger (30 August 2001).

Features

The JOS Technical Edition has many features. It includes a snapshot of mirror directories. It includes a snapshot of JOS Wiki. It includes source code formatted with the WebSource utility. Most of these features are available in the on-line edition.
