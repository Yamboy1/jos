Agents to help serving the user

This page is aimed to give you some overview to recent agent technology, which I wish to incorporate into JOS. Please see my text in ExperimentalGUIGroup (in 1998) about agents.

To get an overview about agent technology, the following links are really good start points:

  • UMBC Agent Web http://agents.umbc.edu/
  • IBM Aglets SDK http://www.trl.ibm.co.jp/aglets/
  • Bits and Pixels Intelligent Agents http://www.bitpix.com/business/main/bitpix.htm
I see one aspect of JOS, that it will (can) serve the user, not (like other OSses today) the other way around.
