Jip Wish List

This is the place to make your wishes known. As before, please sign your comments.

Back to the Jip page.

  • Jip needs to be integrated with the CVS server. Thus, whenever a project is updated on CVS, it is noted in Jip, and whenever a new project is added to Jip, a new module in CVS is automagically created with the proper module administrators (=project owners/developers) to encourage CVS usage and thus better 'publishing' of source code.
      • Jip will be integrated as soon as I have finished the prototype. -- IainLowe (03-02-99)
  • The member portion of Jip needs to be integrated with the jos-vote-info membership information. The jos-vote-info distribution list will define those who are registered as voting members of JOS according to the proposed Constitution.
    • A note in the member listings whether or not a person is a voting member.
    • A link from the member listings/signup to subscribe to the jos-vote-info list and become a voting member of JOS.
      • With regards to this, a module is being developped that will manage the members of the JOS Collective. -- IainLowe (03-02-99)
-- AveryRegier (03 February 1999)
We need a JosControlPanel that will sit on the desktop and do everything that a jos user needs.. (upload files, update a page, change membership info, update project list, vote, etc..) It will be updated with current Jos News and be able to access any data that you can get off the web, but easier...
