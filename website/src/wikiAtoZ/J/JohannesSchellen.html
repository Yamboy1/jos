Hi!

I'm a student of computer science at RWTH-Aachen, Germany. I started programming Java early 1996 and have since written network, user interface and special effect code. I am used to working with Linux, Mac OS and Windows 95.

My main interest would be to help developing the user interface, configuration utilities or a "filemanager" that is worth to be compared to the Mac Finder (and we're not here to be second, are we?).

The most important idea should IMHO be to keep the system well organized and simple (but effective) so that application programmers and users can rely on a robust and understandable system. Oh, and I hate rebooting! Wherever possible things should be installable/changeable without rebooting.

Let the show begin...
