Jamund Iain Peter - Jip Package

All the code for this project is in the org.jos.jip package.

This project is developping a tool for keeping the Projects list up to date and (eventually) for keeping a database of the past and present members of JOS. The database will contain information on each member/project and will allow us to determine the exact status of any project or member at any given time.

JamundFerguson has developped some Perl scripts that will allow us to offer a preliminary implementation of this system

PeterSchuller is working on a Swing-based GUI for this and future "Control Panel"-type applications

IainLowe is working on a 100% Pure Java implementation of the internals of this project.
