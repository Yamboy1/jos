I am planning to check out version "2a" of JJOS and create a new binary edition.

8 February 2002

I have checked out version "2a". I am working toward creating a new binary edition. As explained in JJOSAndBinaryEdition, the target files will have unique names:
  • jjos2a.nbi in the tftpboot directory (Etherboot-compatible)
  • jjos2a-g and jjos2a.zip in the josboot directory (GRUB-compatible)
Also, I am creating a jjos2a directory in the JOS Technical Edition so that we can easily make references to its Java source code.

-- GilbertHerschberger

6 March 2002

I have committed recent changes to version "2a". I am still working toward a binary edition. I have powered up my www box and am compiling the binary edition as I write this.

As I look around for the binary edition of version "1e", I am surprised to find that it has not been completed either.

-- GilbertHerschberger
