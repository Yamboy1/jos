The following article outlines a possible goal of the JOS Project. It is a proposal by GilbertHerschberger (11 August 2001).

Introduction

This helps our members discuss what all Java-based operating systems have in common. Since we are working with common technology, we have common problems. It provides a forum for the free exchange of ideas. It is intended to help both members and non-members to build Java-based operating system. It provides the strongest support and encouragement for free license and open source software. Here is how the project does this.

Promotion

This project maintains an Internet website for the promotion of a Java-based operating system. It explains to the general public what a Java-based operating system might look like and why a new operating system might be desireable.

Members of this project have been researching the idea of a Java-based operating system longer than any other open project. It has identified the most critical issues in building such an operating system. This enables others to immediately start working on issues that has not yet been researched.

This project selected jos.org as its domain and www.jos.org as its website. It remains the premier website for building a Java-based operating system because of the constant, mindful attention of its members.

Delegation

The potential research into a Java-based operating system is far more than most could have imagined. This project began to delegate portions of its research to subprojects, such as Jext, JEPS, sfWiki and many more. Subprojects focus on a specific area of research and provide summary articles for others.

There are so many valid ways to implement a Java-based operating system. This project began to delegate the implementation details to sister projects, such as CJOS, JJOS, kissme and teaseme. A sister project may have enough activity to justify their own mailing lists. Some of their members provide announcements, articles, products and source code. They contribute hands-on discussion of JOS related issues.

While some members argue for a single implementation of a Java-based operating system and single license, this project invites its members to compete on implementation details and software licensing.

Forum

This project provides a forum for our members. We have gathered the best the Internet has to offer to help our members communicate with each other. Today, our preference is to take advantage of both a mailing list and JOS Wiki.
  • jos-general mailing list A mailing list provides a forum for hot topics. It helps keep everyone informed about who's working on what, identifying critical issues and the status of our sister projects. Here is where our members make announcements and discuss issues.
  • JOS Wiki This installation of sfWiki provides a database of articles. Some articles provide project history, how to join and what we've already discussed. It provides many links, links to sister projects, links to subprojects and links to related websites. This forum is designed to educate both members and the general public. This enables our members to find out more about Java technology and its implications for building a Java-based operating system.

Distribution

This project provides an Internet website for the electronic distribution of project-related files. One member uploads files for others to download. This service is intended for the convenience of subprojects, where no alternative is readily available.

Future

The following would be good for this project:
  • More members.
  • More sister projects, with alternative licensing and new ideas for implementation.
  • More subprojects, expanding and building upon the work already done.
