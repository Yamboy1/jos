Introduction

GRUB is a boot loader. The I386Build of JJOS is compatible with it and Etherboot. Using Linux(r) operating system, GRUB can be installed from source code. [A40] This is a three step process:

  1. Download
  2. Compile
  3. Install

1. Download

The first step to install GRUB is to download and untar a source distribution.

  • Make a new directory.

    $ <STRONG>mkdir ~/grub</STRONG>
$ <STRONG>cd ~/grub</STRONG>

  • Download and untar a source distribution. For example, a distribution is available at the following location:

    <STRONG>ftp://alpha.gnu.org/gnu/grub/grub-0.5.95.tar.gz</STRONG>

    After downloading grub-0.5.95.tar.gz, it should be in your ~/grub directory.

  • To untar the source code, use the following Linux(r) command:

    $ <STRONG>tar xvzf grub-0.5.95.tar.gz</STRONG>

    The tar command will create new directories and files in your ~/grub directory.

2. Compile

The second step to install GRUB is to configure the make process for you machine and compile it with the configured make script.

  • Move to your grub-0.5.95 directory.

    $ <STRONG>cd ~/grub/grub-0.5.95</STRONG>

  • To configure the make process, use the following Linux(r) command:

    $ <STRONG>./configure</STRONG>

  • To compile the source code, use the following Linux(r) command:

    $ <STRONG>make</STRONG>

3. Install

The grub program is found in the usr/local/sbin/grub directory. Some of grub's data files are found in the /usr/local/share/grub/i386-pc/ directory. These are well-known directories for GRUB. After downloading and compiling, it needs to be installed in its well-known directories.

  • Switch to the root account. The following installation step will require superuser privileges.

  • Move to the grub-0.5.95 directory, if necessary.

    # <STRONG>cd /home/<EM>user-name</EM>/grub/grub-0.5.95</STRONG>

  • To install GRUB, use the following Linux(r) command:

    # <STRONG>make install</STRONG>

Next

After installing GRUB from source, you might want to create a GRUB Boot Disk. For more information, see the GRUBBootDisk article on JOS Wiki.

Reference

  • The GRUB homepage is http://www.gnu.org/software/grub/ .

  • I recommend you install GRUB like this... -- John Leuner [A40]

With a hint from John Leuner, I installed the linclude package that came with SuSE 6.0 on my Jannux(tm) box (JannuxSuSE). Both the make and make install commands ran without displaying an error message.

On my machine, the make install command copied GRUB files to the following directories:

/usr/local/sbin/grub
/usr/local/shared/i386-pc/grub

The installed grub program is found in the /usr/local/sbin/grub directory. The installed stage1 and stage2 files are found in /usr/local/shared/i386-pc/grub. -- GilbertHerschberger (8 August 2000).
