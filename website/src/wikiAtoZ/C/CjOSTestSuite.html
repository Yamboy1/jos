Introduction

A test program is statically linked to a kernel. A test program is invoked one after the other in a test suite. The name of a test program is used in a menu-driven interface. 
class mc_TestProgram {
  public:
    mc_TestProgram();
    virtual ~mc_TestProgram();

    virtual const char *getName();
    virtual void run() = 0;

  protected:

  private:

}
