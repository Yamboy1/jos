clUtilities1.0.3.zip

CommandLineUtilities 1.0.3 (June 1999) contains several GNU/Linux command line utilities, ported to Java.

  • cal, cat, cd, date, du, echo, ls, mkdir, mv, pwd, rm, rmdir, touch, wc, and whoami; libraries like fnmatch and getopt; and wrappers for javac and jar
  • Localized for several languages
  • Documented with HTML manual pages
  • Distributed as runnable jar files
  • Coming Soon: Available as fully commented source code in SourceServer. And source is now included in the downoad.
