Introduction

The Class Print program has taken a giant leap forward when combined with a microcode approach. It is now able to produce Jasmin-compatible source code from most class files. The microcode approach has greatly reduced the amount of custom coding for individual opcodes.

Package

The following package is part of the CjOS Project CVS.
  • gchii.alt.microcode1a

Classes

The Opcode class represents one opcode.

The OpcodeTable class provides a list of opcodes.

The MicrocodeTable class provides a list of opcodes. It installs microcode information for each opcode.
