Introduction

While the design of java/lang/ClassLoader is better than most mechanisms provided by other technology, it is not as good as it could be. Here is one possible solution.

Class provider

A class provider stores the raw class file for all classes in a package. As a raw class file is converted into an instance of java/lang/Class, the raw class file is garbage collected.

One custom class loader should be able to provide classes in all kinds of custom file formats. Here is an example using local .class files: 

  public void loadPackage( String pn ) {
    String d = separate( codebase + pn2pf( pn ) );
    File f = new File( d );
    String[] list = f.list( classFilter );
    int iMax = list.length;
    for ( int i = 0; i < iMax; i++ ) {
      String cn = pn + "." + cf2cn( list[ i ] );
      cache.put( cn, grab( d + list[ i ] ) );
    }
  }

Interface

The java/lang/ClassLoader class should have been a well-known interface or should have implemented a well-known interface. 
public class ClassFinder {
  /**
   * Gets ClassFor[] property.
   */
  public abstract Class getClassFor( String cn );
}
Here is a snippet of the source code for a new java/lang/ClassLoader class. 
public class ClassLoader

getClassFor() method

This method returns an instance of java/lang/Class for the given class name. A class name always uses the dot-notation, such that the class name for java/lang/Object is java.lang.Object.

Extension

Many custom classes are capable of returning a class for a given class name. The ClassFinder interface should be useful. A custom class loader -- even in Java 0, 1 and 2 -- can implement a well-known interface to find classes. 
  public void example() {
    String cn = "org.jos.demo.DemoClass";
    Class c = findClass( cn );
  }
  protected Class findClass( String v ) {
    ClassLoader loader;

    {
      /*
      When loading a custom class using the same class
      loader as this class.
      */
      loader = getClass().getClassLoader();
      if ( loader instanceof ClassFinder ) {
        Class c = ((ClassFinder) loader).getClassFor( v );
        if ( c != null ) {
          return c;
        }
      }
    }

    {
      /*
      When loading a custom thread with a custom
      class loader.
      */
      Thread t = Thread.currentThread();
      loader = t.getClass().getClassLoader();
      if ( loader instanceof ClassFinder ) {
        Class c = ((ClassFinder) loader).getClassFor( v );
        if ( c != null ) {
          return c;
        }
      }
    }

    /*
      When loading a system class.
    */
    try {
      return Class.forName( v );
    }
    catch ( Throwable e ) {
      return null;
    }
  }
