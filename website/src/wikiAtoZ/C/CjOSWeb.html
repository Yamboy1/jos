Introduction

Take the CjOS Project for a test drive.

This is a new comprehensive product for developers and end-users. It enables you to use all of the CjOS utilities on your own computer...

  • without compiling a line of code,
  • without creating a custom batch file on Microsoft Windows,
  • without using a CVS-compatible front-end, and
  • without missing the latest and greatest releases.

Pre-compiled

Every CjOS package is pre-compiled into class files. In turn, class files are archived into package files.
  • Note: When a Java package changes, the Erica tool downloads the corresponding package file so you'll never miss the latest and greatest releases.

Integrated into Microsoft Windows

The eric tool for Windows integrates this suite of Java applications into the Microsoft Windows platform. It runs on Windows 95/98/me and Windows NT 4.0, 5.0 (2000) and 5.1 (XP).

Distributed across HTTP

This product and its packages are distributed across the HTTP protocol. The complexity of CVS has been avoided. And yet, the Erica tool downloads packages as needed so you never need to install the product again.
