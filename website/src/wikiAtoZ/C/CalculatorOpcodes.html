Introduction

Calculator opcodes are a subset of all opcodes, the instruction set of a virtual machine. These opcodes might make use of a separate calculator library, part of a VMKit.

Integer Table

The following table is a partial list of integer-related opcodes and their corresponding calculator function prototypes.

Expression Opcode Prototype
r += a; iadd int iadd( jint a, jint &r );
r -= a; isub int isub( jint a, jint &r );
r *= a; imul int imul( jint a, jint &r );
r /= b; idiv int idiv( jint a, jint &r );
r %= a; irem int irem( jint a, jint b, jint &r );
r = -r; ineg int ineg( jint &r );
r = r << 1; ishl int ishl( jint &r );
r = r >> 1; ishr int ishr( jint &r );
r = r >>> 1; iushr int iushr( jint &r );
r &= a; iand int iand( jint a, jint &r );
r |= a; ior int ior( jint a, jint &r );
r ^= a; ixor int ixor( jint a, jint &r );
r++; iinc int iinc( jint &r );
r = (int) a; l2i int l2i( jlong a, jint &r );
r = (int) a; f2i int f2i( jfloat a, jint &r );
r = (int) a; d2i int d2i( jdouble a, jint &r );
r = (byte) r; i2b int i2b( jint &r );
r = (char) r; i2c int i2c( jint &r );
r = (short) r; i2s int i2s( jint &r );
