Introduction

While working on a keyboard interrupt service routine, I stumbled upon that thing we call a "circular queue". This queue shows up frequently when there are limited resources and asynchronous requests coming in.

After a bit of refinement, I came up with this. It is a node for a circular queue in Java.

public class Node {
  public Node() {
	 this( null );
  }
  public Node( Node v ) {
	 setNext( v );
  }
  public Node getNext() {
	 return next;
  }
  public void setNext( Node v ) {
	 next = v;
  }
  public Object getData() {
	 return data;
  }
  public void setData( Object v ) {
	 data = v;
  }
  private Node next;
  private Object data = null;
}

Example

Here is an example program that builds a circular queue.

public class Example {
  public static void main( String[] args ) {
	 Node head = new Node();
	 Node tail =
		  new Node( new Node( new Node( head ) ) );
	 head.setNext( tail );
	 
	 for ( int i = 0; i < 22500; i++ ) {
		head = head.getNext();
		head.setData( new Integer( i ) );
	 }
  }
}
