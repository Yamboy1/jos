Introduction

CJOS CVS on SourceForge refers to a part of SourceForge reserved for files from the CJOS Project available through a CVS prototol. This is a thing that I keep referring to, but have not defined. This has become an important part of the CJOS Project.

I would like to define it here. The folks at SourceForge were kind enough to let us open source programmers use their equipment and services. The CJOSProject is an open source project. The CVSService enables contributors to upload their files. A CVSClient enables others to download these files from a server. Here, a CVS service runs on servers at SourceForge and a CVS client runs on a machine connected to the Internet.

Migration

The java0 and java1 modules must not contain source code in the com.realignsoft namespace. That namespace must no longer be used for our open source project. It is necessary to migrate packages from com.realignsoft to gchii.library, gchii.si or org.jos. The following packages have been moved:
  • org.jos.smartapi1a (was com.realignsoft.smartapi1a)
