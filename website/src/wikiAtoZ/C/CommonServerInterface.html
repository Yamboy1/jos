Common Server Interface (CSI)

Latest revision: 0.4 / Apr 7, 1998
Official site: http://www.jos.org/wiki/view/main/CommonServerInterface
Download from: ftp://ftp43.pair.com/pub/losgann/csi/csi-0.4.zip

CSI (or Common Server Interface) is a set of components usable by internet protocols daemons (or possibly any daemons for that matter) which allows for configuration, user interface, kernel interface, network interface, etc. in a common fashion. Contribute if you like. Most recent information will be at above site in the Java Development section.
