Section 4 has only one possible problem. 8 spaces instead of 4 seems wasteful, and besides, that's for XEmacs, Symantec Cafe, FreeBuilder, or edit.com to decide. If tabs aren't set in an editor, four presses on the spacebar seems more useful than eight. Many other editors / development companies have standardized on 3 spaces for the tab key.
