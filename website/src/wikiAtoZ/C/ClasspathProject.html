Introduction

The classpath project is part of GNU and found at http://www.classpath.org

Relationship with JOS Project

The JOS Project and the classpath Project are developing a Java-compatible virtual machine. Both project are open-source, free-license software projects on the Internet. These projects are similar, but not the same. The classpath project is working to create a classic Java-compatible virtual machine. They are sticking close to the architecture of "JVM on foreign OS", just like a JDK/JRE from Sun Microsystems.

As members of the JOS Project, we are working to create a new kind of Java-compatible virtual machine. We are not sticking so close to previous architecture. Instead of "JVM on foreign OS", we're working on something like a "JVM doesn't need a foreign OS". We look forward to using the pre-compiled bytecode from the classpath project in the free-license distribution of JOS.
