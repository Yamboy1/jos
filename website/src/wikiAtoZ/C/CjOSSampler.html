CjOSPages

Article contributed by GilbertHerschberger (6 December 2001).

Introduction

The CjOS Sampler enables someone to see how applets might look in a webpage using your favorite browser. For the HTML author, it provides examples of using a Java applet on a page.
  • http://cjos.sourceforge.net/sampler/

Sampler

To demonstrate what you know about including a Java applet on an HTML page, you should create a sampler. A sampler provides evidence that your HTML page works in a brower and an applet works on that HTML page. There may be no better way to prove that you have a handle on Java applets like a sampler.

Work for hire

I started writing HTML pages a few years ago. After getting a handle on HTML, I started writing Java applets. The best examples of my work is not available for public view because it was part of my client's intranet. As "work for hire", I must not upload this to the Internet, or expose it to public view.

For public view

Unlike "work for hire", this sampler is intended for public view. It includes all new HTML pages describing the applets themselves and how they work. This will soon be installed on the Internet.

About BulkMake

The Package Archive utility is part of the BulkMake product distribution. It manages the complexity the jar tool. A new version of the Package Archive utility is required in order to properly archive classes for the CjOS Sampler. The sampler needs uncompressed ZIP-compatible files and the new version will be capable of building them. (If possible, it will also be capable of including an external manifest file.)
