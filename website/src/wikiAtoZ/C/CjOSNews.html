1 January 2002

  • A number of releases need to be distributed.
    • Release 5 of Smart API 1a needs to be distributed. Created the jos1m-smartapi1a.zip and jos1m-smartapi1a-src.zip archives.
    • Release 6 of Smart API 1b needs to be distributed. Created the jos1m-smartapi1b.zip and jos1m-smartapi1b-src.zip archives.
    • Release 18 of Smart API 2a needs to be distributed. Created the jos1m-smartapi2a.zip and jos1m-smartapi2a-src.zip archives.
    • Release 2 of Khimera 1d needs to be distributed. Created the jos1m-khimera1d.zip archive. For more information, see also AboutKhimera. Upgraded test programs from TestApplication to TestProgram.
  • Upgraded Wiki Changes from version 1a to 1b. The new version preserves the original article names and is implemented in the gchii.wiki.changes1b package. Used with sfWiki software.
-- GilbertHerschberger

26 December 2001

  • A number of products need to be re-released.
    • Smart API 1a - This version has been moved to org.jos.smartapi1a from com.realignsoft.smartapi1a. Plug-ins for this version are no longer implemented in the com.realignsoft namespace.
    • Smart API 1b - This version has been moved to org.jos.smartapi1b from com.realignsoft.smartapi1b. Plug-ins for this version are no longer implemented in the com.realignsoft namespace.
    • Smart API 2a - This version was always org.jos.smartapi1b; but, its plug-ins are no longer implemented in the com.realignsoft namespace.
    • The WebSource (tm) utility is distributed as an independent product. It converts Java source code into HTML pages for easy browsing.
  • The Package Compiler utility is package-oriented and part of BulkMake. The next version of this product should use a checkbox component for the show property, instead of a text component. The only value of the show property turns out to be all (or nothing).
-- GilbertHerschberger

6 December 2001

  • The Alternative Bridge (AltBridge) product, written in the Java programming language, is now part of the CjOS build module. Look for the altbridge directory.
  • Both the limited and unlimited builds from the ERIC Project, written in the Java programming language, are now part of the CjOS build module. Look for the ericlimited and ericunlimited directories.
  • The applets from the CjOS Sampler, written in the Java programming language, are now part of the CjOS build module. Look for the sampler directory.
  • The first version of CjOS Sampler has been commit'ed to CVS. Look for the sampler1 module. For more information, see also the CjOSSampler article on JOS Wiki.
  • Release 9 of ERIC 1c has been commit'ed to CVS. Look for the cpp/eric1c module. This release prepares for switching CharBuffer to version 1b. For more information, see also the release1c-9.txt file in the doc directory.
-- GilbertHerschberger
