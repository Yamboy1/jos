Application: ResourceUtility

Type: Utility

Package: org.jos.util

Source Code: http://www.jos.org/source/org/jos/util/ResourceUtility.java.html

Objective: Implement internationalisation issues very easily into any application.

This utility contains the following features:

  • Uses default and any other stated locale as base
  • Retrieves strings from ResourceBundle
  • Sets up a Frame object according to values given in the ResourceBundle

Static methods to retrieve:

  • Image object from a file name
  • Mnemonics from text such as 'sAve as' - would use 'A' as the mnemonic
  • TitleCase would convert 'sAve as' to 'Save As'
  • List of items seperated by whitespaces to an array of Strings
  • Creates a JLable instance using a resource string ('sAve as')
