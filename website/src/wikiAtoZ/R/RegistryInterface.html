RegistryDesign

Article contributed by GilbertHerschberger.

Introduction

The registry interface is divided into five parts.

  1. KeyedItemAccessor
    The keyed item accessor methods look like this: 
    boolean getBoolean( String key );
byte getByte( String key );
char getChar( String key );
short getShort( String key );
int getInt( String key );
long getLong( String key );
float getFloat( String key );
double getDouble( String key );
String getString( String key );
  2. KeyedItemModifier
    The keyed item modifier methods look like this: 
    void setBoolean( String key, boolean v );
void setByte( String key, byte v );
void setChar( String key, char v );
void setShort( String key, short v );
void setInt( String key, int v );
void setLong( String key, long v );
void setFloat( String key, float v );
void setDouble( String key, double v );
void setString( String key, String v );
  3. KeyedArrayAccessor
    The keyed array accessor methods look like this: 
    boolean[] getBooleanArray( String key );
byte[] getByteArray( String key );
char[] getCharArray( String key );
short[] getShortArray( String key );
int[] getIntArray( String key );
long[] getLongArray( String key );
float[] getFloatArray( String key );
double[] getDoubleArray( String key );
String[] getStringArray( String key );

  4. KeyedArrayModifier
    The keyed item modifier methods look like this: 
    void setBooleanArray( String key, boolean[] v );
void setByteArray( String key, byte[] v );
void setCharArray( String key, char[] v );
void setShortArray( String key, short[] v );
void setIntArray( String key, int[] v );
void setLongArray( String key, long[] v );
void setFloatArray( String key, float[] v );
void setDoubleArray( String key, double[] v );
void setStringArray( String key, String[] v );
  5. Registry
    The registry-specific methods look like this: 
    String[] getKeys();
String getType( String key );
Object getObject( String key );
void setObject( String key, Object v );
Registry getRegistry( String key );
void setRegistry( String key, Registry v );
Registry[] getRegistryArray( String key );
void setRegistryArray( String key, Registry[] v );
void saveAs( String url );
