Introduction

The i386 build is compatible with both Etherboot and GRUB. GRUB has been growing in popularity among members of the JOS Project over the last six months. GRUB is now the preferred method.

I set a goal for myself some time ago. I want to boot JOS using GRUB. I'd like to create a boot-able CD-ROM. This article is especially a work-in-process. I have more questions than answers. If you could answer a question or ask something new, please send me an e-mail at gchii@mindspring.com . Thank you.

How it works

Move to your JOS machine, the machine that actually runs this operating system.

  1. Insert your JOS Boot Disk.

  2. Press the reset button.

  3. Your JOS machine invokes GRUB.

  4. GRUB loads a boot image from your hard drive (or CD-ROM).

  5. GRUB jumps into the boot image. This is how JOS boots with GRUB.

Requirements

Unlike Etherboot, a network interface card is optional. Otherwise, you must choose to share or not to share. This shapes your hardware requirements.

  • Share

    It is possible to share one machine between Linux and JOS. You boot this machine with GRUB only when you want to use JOS.

    • The requirements for Linux will have the greatest impact on what kind of machine it must be.

    • When you use one machine for Linux and JOS, you'll propably boot JOS from a floppy or CD-ROM. You might need a floppy drive. You might need a CD-ROM drive.

  • Not to share

    A low-cost JOS machine is a good idea. It protects the important data on your Linux hard drive. You can put together a low-cost JOS machine at a "previously-owned computer" store. Look for a '486/66MHz machine with 16MB RAM, a floppy and any hard drive. Last week (1 March 2000), I bought one for US$40. (It even had a sound card and CD-ROM.)

    • jJOS/decaf is about 4,000,000 bytes. You'll need a hard drive or CD-ROM to complete the boot process.

    • When you use a machine exclusively for JOS, you can boot JOS from a hard drive or CD-ROM. You might not need a floppy.

Installation

You must install a boot image and create your own JOS Boot Disk. Move to your Linux machine

  1. Boot with Linux.

  2. Unzip or untar the source distribution of the JJOS module.

  3. Configure the build process for your foreign JDK.

  4. Run build (or make) to create JVM-specific classes and a i386 build.

  5. Install (copy) the i386 boot image where GRUB can find it.

  6. Create your own JOS Boot Disk. Copy GRUB and its boot configuration to it.
