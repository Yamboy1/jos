CORBA and Java are virtually perfect for one another. The Interface Definition Language meshes extremely closely with Java's notion of interfaces and coding in IDL is familiar enough to anyone who has done any C/C++/Java (in fact, Sun is a member of the OMG who developed CORBA). If anyone is interested in Programming Java/CORBA, I would suggest CLIENT/SERVER PROGRAMMING WITH JAVA AND CORBA...I don't know the ISBN and the author's names are a bit weird, but Amazon.com should have no problems with the title (it's currently number fifty on their "hot" list). The OMG's website (http://www.omg.org/) has plenty of references and tutorials. I've also downloaded a free Java/CORBA orb called appropriately Jacorb...but I think the ORB is not CORBA 2.0 (I haven't had much chance to work with it). ========JamesRichards========
